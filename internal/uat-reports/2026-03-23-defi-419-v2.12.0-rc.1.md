## PASS: 3/6

### defi-09 (Pendle buy_pt) — R4에서 PASS
- **Root cause**: Pendle API 응답 형식 3건 변경
  1. `/v2/sdk/{chainId}/convert` 응답이 `{ routes: [{ tx, outputs }] }` 형식으로 변경
  2. `/v1/markets/all` 응답이 `{ markets: [...] }` 래퍼 형식으로 변경
  3. `underlyingAsset` 필드가 object → string으로 변경 (`"1-0xabc..."`)
  4. `pt`/`yt`/`sy` 필드에 chainId prefix 추가 (`"1-0xabc..."`)
- **Fix**: PendleConvertRoutesSchema 추가, PendleMarketsResponseSchema에 `markets` 키 추가, PendleUnderlyingAssetSchema union, stripChainPrefix transform

### defi-14 (DCent get_quotes) — R2에서 PASS
- **Root cause**: ApiDirectResult의 `data` 필드가 응답 직렬화에서 누락
- **Fix**: `actions.ts` 4곳에서 `data` 필드 포함하도록 수정

## FAIL: 2/6 (환경 의존)

### defi-08 (Kamino) / defi-10 (Drift) — RPC 429
- **Root cause**: Solana public RPC 전체 429 rate limit
- **Code fix applied**: KaminoSdkWrapper/DriftSdkWrapper에 URL resolver 함수 패턴 적용, `registerBuiltInProviders`에 `solanaRpcResolver` 옵션 추가하여 RpcPool 연동
- **Verdict**: 코드 수정 완료. 유료 RPC endpoint 설정 시 동작 예상. 현재 환경에서는 public endpoint만으로는 통과 불가.
- **Action**: 유료 Solana RPC (Helius, QuickNode 등) 설정 후 재검증 필요

## PARTIAL/FAIL: 2/6 (DCent API 제한)

### defi-15 (DCent Crosschain) — DCent API 미지원
- **Root cause**: DCent Swap API가 EVM→Solana 크로스체인 라우트에 대해 empty txdata 반환
- **Debug dump**: `~/.waiaas/logs/dcent-debug/` 에 요청/응답 JSON 저장됨
- **Verdict**: WONTFIX — DCent API 자체 제한. 디센트팀에 디버그 로그 전달 필요

### defi-16 (DCent Solana Swap) — 시뮬레이션 코덱 에러
- **Root cause**: DCent API가 Solana 스왑용 serialized transaction을 반환하지만, 시뮬레이션 시 `fixCodecSize` 에러 발생 (서명 슬롯 크기 불일치 — 44/47 bytes vs expected 64 bytes)
- **Code fix applied**: SolanaAdapter.buildContractCall에 pre-built serialized transaction bypass 추가
- **Verdict**: dryRun 시뮬레이션만 실패. 실제 실행(signing) 시에는 동작할 가능성 있음. DCent 반환 트랜잭션 형식이 @solana/kit 6.x 코덱과 호환되지 않을 수 있음. 디센트팀 확인 필요.
