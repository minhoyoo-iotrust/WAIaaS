## Failed Scenarios

### defi-12: ETH→USDC 메인 스왑
- **Failed Step**: Step 2 (DCent 스왑 Simulate)
- **Error**: `execution reverted` — DCent 라우터 `0xAC4c...0b75`에서 revert
- **Root Cause**: DCent API `txdata.value`가 `10925036` wei (프로토콜 수수료)만 반환. 스왑 금액 `5000000000000000` wei (0.005 ETH)가 `msg.value`에 포함되지 않아 라우터에서 ETH 수령 불가
- **Code**: `dex-swap.ts:264` — `txdata.value`를 그대로 사용, 네이티브 매도 시 value 보정 없음
- **Issue**: #393

### defi-12b: 크로스체인 EVM→Solana
- **Failed Step**: Step B1 (EVM→Solana Simulate)
- **Error**: `DCent API returned empty txdata`
- **Root Cause**: 미확인. DCent Swap Aggregator는 ButterSwap/Rubic 등을 통해 EVM↔Solana 크로스체인을 지원하므로, 요청 파라미터 매핑/소액 최소 금액 미달/프로바이더 선택 문제 가능성
- **Note**: 추가 디버깅 필요 — API 요청/응답 상세 로그 확인

### defi-12c: Solana SOL→USDC
- **Failed Step**: Step C1 (SOL→USDC Simulate)
- **Error**: Zod validation — `txdata.from: Required`, `txdata.to: Required`
- **Root Cause**: `DcentTxDataResponseSchema`가 EVM 전용 (`from`/`to`/`data`). Solana 트랜잭션은 serialized bytes로, EVM 구조와 불일치. `dex-swap.ts`에 체인 분기 없음
- **Code**: `schemas.ts:73-77` (EVM 전용 스키마), `dex-swap.ts:259-265` (EVM 전용 실행 로직), `index.ts:86` (`chains: ['ethereum', 'solana']` 선언과 불일치)
- **Issue**: #394
