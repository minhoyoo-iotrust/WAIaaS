# UAT 실행 리포트 영구 보관 시스템

- **유형:** ENHANCEMENT
- **심각도:** MEDIUM
- **마일스톤:** v31.17
- **상태:** FIXED

## 설명

Agent UAT 실행 결과를 `internal/uat-reports/` 디렉토리에 마크다운 파일로 저장하여 버전별 테스트 이력을 추적한다. 리포트는 Git에 커밋되므로 지갑 주소, 세션 토큰 등 민감 정보를 마스킹해야 한다.

## 현재 문제

- UAT 실행 결과가 대화 컨텍스트 안에서만 존재하고 영구 보관되지 않음
- 어떤 버전에서 어떤 테스트를 돌렸는지, 결과가 어떠했는지 추적 불가
- 회귀 테스트 비교 기준 데이터 부재

## 구현 항목

### 1. `internal/uat-reports/README.md` 생성

- 디렉토리 목적 및 파일명 규칙 정의
- 파일명 포맷: `{YYYY-MM-DD}-{category|scenario-id}-v{version}.md`
- 동일 날짜 재실행 시 `-{n}` 접미사

### 2. 프라이버시 마스킹 규칙

| 항목 | 마스킹 규칙 | 예시 |
|------|------------|------|
| EVM 주소 | 앞 6자 + `...` + 뒤 4자 | `0x1a2B...9c0D` |
| Solana 주소 | 앞 4자 + `...` + 뒤 4자 | `7xKX...m9Fp` |
| TX 해시 | 앞 10자 + `...` + 뒤 6자 | `0x3f8a1b...a1b2c3` |
| 세션/마스터 토큰 | **절대 포함 금지** | — |
| API 키 | **절대 포함 금지** | — |
| Wallet ID | 그대로 노출 (내부 UUID) | `wallet-abc123` |

### 3. 리포트 포맷

필수 섹션:
- 헤더: 날짜, 버전(commit short hash 포함), 카테고리, 네트워크, 실행자
- Summary 테이블: Total/Passed/Failed/Skipped/Total Gas Cost
- Results 테이블: 시나리오별 ID, 제목, 상태, 가스비, 소요시간, 비고
- Failed Scenarios: 실패 Step, 에러 메시지, 시도한 해결 방법
- Environment: 데몬 포트, RPC (호스트만, API 키 제외), OS, Node 버전

### 4. SKILL.md Phase 8 업데이트

현재 Phase 8(리포트 출력)에 리포트 파일 저장 로직 추가:
- 리포트 마크다운 생성 시 프라이버시 마스킹 적용
- `internal/uat-reports/` 디렉토리에 파일 저장
- 기존 파일명과 중복 시 `-{n}` 접미사 자동 부여
- 저장 완료 후 파일 경로를 사용자에게 안내
