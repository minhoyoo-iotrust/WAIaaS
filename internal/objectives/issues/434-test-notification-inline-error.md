# 434 — Admin UI: 테스트 알림 실패 시 에러 메시지가 눈에 띄지 않음

- **유형:** ENHANCEMENT
- **심각도:** LOW
- **상태:** OPEN
- **등록일:** 2026-03-24
- **파일:** `packages/admin/src/pages/human-wallet-apps.tsx`

## 설명

지갑 앱 카드에서 Test 버튼을 눌러 테스트 알림을 보낼 때, 실패 에러가 우측 상단 토스트로만 표시된다.
토스트는 5초 후 자동 소멸하고 화면 모서리에 위치해 눈에 잘 띄지 않는다.

## 개선 방안

- 테스트 실패 시 해당 카드 내부에 인라인 에러 메시지를 직접 표시한다.
- Test 버튼 옆이나 바로 아래에 빨간색 텍스트로 에러 사유를 보여준다.
- 성공 시에는 기존 토스트를 유지하거나 인라인 성공 메시지로 대체한다.

## 테스트 항목

- [ ] Test 버튼 클릭 후 실패 시 카드 내 인라인 에러 메시지가 표시되는지 확인
- [ ] 에러 메시지가 토스트보다 가시성이 높은지 확인
- [ ] 성공 시 적절한 피드백이 표시되는지 확인
- [ ] 다른 카드의 에러 상태와 독립적으로 동작하는지 확인
