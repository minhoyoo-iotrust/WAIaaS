# Plan 281-02: MCP Tool Deletion + Description Update

## Goal
Delete the `set_default_network` MCP tool and update all 24 remaining MCP tool descriptions to replace "default wallet/default network" references with explicit specification guidance.

## Context

Phase 280 deleted the `PUT /v1/wallet/default-network` endpoint, making the `set_default_network` MCP tool non-functional. Additionally, all 24 other MCP tools have `wallet_id` descriptions saying "Omit to use the default wallet" and `network` descriptions saying "Defaults to wallet default network" -- both phrases are no longer accurate since the default wallet/default network concepts are removed.

**New description patterns:**
- `wallet_id`: `'Target wallet ID. Required for multi-wallet sessions; auto-resolved when session has a single wallet.'`
- `network` (for transaction tools): `'Target network (e.g., polygon-mainnet). Required for EVM wallets; auto-resolved for Solana.'`
- `network` (for query tools with 'all' support): `"Query balance for specific network. Use 'all' for all networks. Required for EVM wallets; auto-resolved for Solana."`
