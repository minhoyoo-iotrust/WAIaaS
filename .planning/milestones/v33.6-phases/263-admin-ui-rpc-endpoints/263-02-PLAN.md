## Enhance each URL row with live status indicator

In the RpcEndpointsTab component (modified by Plan 01), add:

1. **Pool status polling signal:**
   ```ts
   const rpcPoolStatus = useSignal<Record<string, RpcEndpointStatus[]>>({});
   ```
   Where `RpcEndpointStatus = { url: string; status: 'available' | 'cooldown'; failureCount: number; cooldownRemainingMs: number }` (mirror the backend type).

2. **Periodic fetch** via useEffect:
   - On mount and every 15 seconds, call `GET /admin/rpc-status` via apiGet
   - Store result in `rpcPoolStatus` signal
   - Clear interval on unmount
   - Silent failures (don't show toast on polling errors)

3. **Status display per URL row:**
   For each URL in a network's list, look up its status from `rpcPoolStatus.value[network]`:
   - If status === 'available': show green dot + "Available" text
   - If status === 'cooldown': show orange/red dot + "Cooldown" text + remaining time formatted as "Xm Ys remaining" + failure count badge
   - If not found in pool status (e.g. URL just added, not yet in pool): show gray dot + "Unknown"

4. **Per-URL Test button:**
   Each URL row gets a "Test" button (using existing Button component with variant="ghost" size="sm"):
   - On click: determine chain from network key (mainnet/devnet/testnet -> 'solana', everything else -> 'evm')
   - Call `POST /admin/settings/test-rpc` with `{ url, chain }`
   - Track loading state per URL: `rpcTesting` signal `Record<string, boolean>`
   - Track results per URL: `rpcTestResults` signal `Record<string, RpcTestResult>`
   - Display result inline below/beside the URL:
     - Success: green Badge "OK" + latency in ms + block number
     - Failure: red Badge "FAIL" + error message

5. **Result display format** (reuse existing `rpc-test-result` CSS classes from the old tab):
   ```tsx
   {testResult && (
     <span class={`rpc-test-result ${testResult.success ? 'rpc-test-result--success' : 'rpc-test-result--failure'}`}>
       <Badge variant={testResult.success ? 'success' : 'danger'}>
         {testResult.success ? 'OK' : 'FAIL'}
       </Badge>
       {testResult.latencyMs}ms
       {testResult.blockNumber !== undefined && ` (block #${testResult.blockNumber.toLocaleString()})`}
       {testResult.error && ` - ${testResult.error}`}
     </span>
   )}
   ```

6. **Cooldown remaining time display helper:**
   ```ts
   function formatCooldown(ms: number): string {
     if (ms <= 0) return '';
     const sec = Math.ceil(ms / 1000);
     if (sec < 60) return `${sec}s`;
     const min = Math.floor(sec / 60);
     const remSec = sec % 60;
     return remSec > 0 ? `${min}m ${remSec}s` : `${min}m`;
   }
   ```
