1. `packages/e2e-tests/src/helpers/http-client.ts` 생성:

   ```typescript
   /**
    * E2E HTTP 클라이언트 — Node fetch 래퍼.
    * 인증 토큰 자동 첨부, JSON 요청/응답 편의 메서드 제공.
    */
   export class E2EHttpClient {
     constructor(
       private baseUrl: string,
       private token?: string,
     ) {}

     /** 토큰 설정 (세션 생성 후 호출) */
     setToken(token: string): void;

     /** 토큰 제거 */
     clearToken(): void;

     /** GET 요청 */
     async get<T = unknown>(path: string, opts?: { headers?: Record<string, string> }): Promise<{
       status: number;
       body: T;
       headers: Headers;
     }>;

     /** POST 요청 (JSON body) */
     async post<T = unknown>(path: string, body?: unknown, opts?: { headers?: Record<string, string> }): Promise<{
       status: number;
       body: T;
       headers: Headers;
     }>;

     /** PUT 요청 (JSON body) */
     async put<T = unknown>(path: string, body?: unknown, opts?: { headers?: Record<string, string> }): Promise<{
       status: number;
       body: T;
       headers: Headers;
     }>;

     /** DELETE 요청 */
     async delete<T = unknown>(path: string, opts?: { headers?: Record<string, string> }): Promise<{
       status: number;
       body: T;
       headers: Headers;
     }>;

     /** PATCH 요청 (JSON body) */
     async patch<T = unknown>(path: string, body?: unknown, opts?: { headers?: Record<string, string> }): Promise<{
       status: number;
       body: T;
       headers: Headers;
     }>;

     /** 내부: fetch 래퍼 */
     private async request<T>(method: string, path: string, body?: unknown, opts?: { headers?: Record<string, string> }): Promise<{
       status: number;
       body: T;
       headers: Headers;
     }>;
   }
   ```

   - request() 구현:
     - URL: `${this.baseUrl}${path}`
     - Headers: `Content-Type: application/json` (body가 있을 때)
     - `this.token`이 있으면 `Authorization: Bearer ${this.token}` 자동 추가
     - opts.headers로 추가 헤더 병합
     - 응답 body: `res.json()` (Content-Type이 application/json인 경우), 아니면 text()를 body로
     - 204 No Content 등 body 없는 경우 null 반환

2. `packages/e2e-tests/src/helpers/session.ts` 생성:

   ```typescript
   /**
    * 세션 관리 유틸리티 — 마스터 패스워드 설정 + 세션 생성을 한 번에 수행.
    */
   export class SessionManager {
     private client: E2EHttpClient;
     private _token?: string;

     constructor(baseUrl: string) {
       this.client = new E2EHttpClient(baseUrl);
     }

     /** 마스터 패스워드 설정 (최초 1회) */
     async setMasterPassword(password: string): Promise<void>;

     /** 세션 생성 → 토큰 저장 → HTTP 클라이언트에 토큰 설정 */
     async createSession(password: string): Promise<string>;

     /** 세션 갱신 */
     async renewSession(): Promise<string>;

     /** 세션 삭제 */
     async deleteSession(): Promise<void>;

     /** 인증된 HTTP 클라이언트 반환 */
     get http(): E2EHttpClient;

     /** 현재 토큰 */
     get token(): string | undefined;
   }

   /**
    * 한 줄 편의 함수: 마스터 패스워드 설정 + 세션 생성 + 인증된 HTTP 클라이언트 반환.
    *
    * @example
    * const { session, http } = await setupDaemonSession(baseUrl);
    * const { body } = await http.get('/v1/wallets');
    */
   export async function setupDaemonSession(
     baseUrl: string,
     password?: string,
   ): Promise<{ session: SessionManager; http: E2EHttpClient; token: string }>;
   ```

   - setMasterPassword: PUT /v1/auth/master { password }
   - createSession: POST /v1/auth/session { password } → 200 { token } → this.client.setToken(token)
   - renewSession: POST /v1/auth/session/renew → 200 { token } → 토큰 업데이트
   - deleteSession: DELETE /v1/auth/session → 200
   - setupDaemonSession: new SessionManager → setMasterPassword → createSession → { session, http, token } 반환
   - password 기본값: `'e2e-test-password-12345'` (DaemonManager와 동일)

   **빌드 여부 체크**: CLI 빌드 미완료 시 skip (Plan 02와 동일 패턴).

   ```typescript
   describe('Session + HTTP helpers', { timeout: 30_000 }, () => {
     let daemon: DaemonManager;
     let instance: DaemonInstance;

     beforeAll(async () => {
       daemon = new DaemonManager();
       instance = await daemon.start();
     });

     afterAll(async () => {
       await daemon.stop().catch(() => {});
     });

     it('setupDaemonSession: sets master password and creates session in one call', async () => {
       const { session, http, token } = await setupDaemonSession(
         instance.baseUrl,
         instance.masterPassword,
       );

       expect(token).toBeTruthy();
       expect(typeof token).toBe('string');
       expect(token.split('.').length).toBe(3); // JWT format

       // 인증된 API 호출 테스트
       const { status, body } = await http.get('/v1/wallets');
       expect(status).toBe(200);
       expect(body).toHaveProperty('wallets');
     });

     it('HTTP client auto-attaches auth token', async () => {
       const { http } = await setupDaemonSession(
         instance.baseUrl,
         instance.masterPassword,
       );

       // 토큰 없이 요청하면 401
       const noAuth = new E2EHttpClient(instance.baseUrl);
       const { status: unauthed } = await noAuth.get('/v1/wallets');
       expect(unauthed).toBe(401);

       // 토큰 있으면 200
       const { status: authed } = await http.get('/v1/wallets');
       expect(authed).toBe(200);
     });

     it('session lifecycle: create -> renew -> delete', async () => {
       const { session } = await setupDaemonSession(
         instance.baseUrl,
         instance.masterPassword,
       );

       // Renew
       const newToken = await session.renewSession();
       expect(newToken).toBeTruthy();
       expect(newToken).not.toBe(session.token); // 갱신된 토큰은 이전과 다름 (대부분의 경우)

       // 갱신된 토큰으로 API 호출
       const { status } = await session.http.get('/v1/wallets');
       expect(status).toBe(200);

       // Delete
       await session.deleteSession();
     });
   });
   ```

   **주의**: 데몬은 마스터 패스워드를 한 번만 설정할 수 있으므로, 여러 테스트에서 같은 데몬 인스턴스를 공유할 때 setMasterPassword가 이미 설정된 경우를 처리해야 한다. setupDaemonSession에서 PUT /v1/auth/master가 409(이미 설정됨)를 반환하면 무시하고 진행하도록 SessionManager.setMasterPassword 구현에 반영.
