1. `packages/e2e-tests/src/helpers/daemon-lifecycle.ts` 생성:

   ```typescript
   export interface DaemonInstance {
     dataDir: string;
     port: number;
     baseUrl: string;
     process: ChildProcess;
     masterPassword: string;
   }

   export class DaemonManager {
     private instance: DaemonInstance | null = null;

     /**
      * 데몬을 기동한다.
      * 1. os.tmpdir()에 임시 데이터 디렉토리 생성 (waiaas-e2e-{uuid})
      * 2. keystore/, data/, logs/, backups/ 하위 디렉토리 생성
      * 3. 빈 포트를 찾아 config.toml 작성 ([daemon] port, hostname=127.0.0.1, [database] path=data/waiaas.db)
      * 4. child_process.fork()로 CLI 엔트리포인트 실행
      *    - fork 대상: 모노레포 내 packages/cli/bin/waiaas (resolve via require.resolve or path.resolve)
      *    - args: ['start', '--data-dir', dataDir]
      *    - env: { ...process.env, WAIAAS_MASTER_PASSWORD: masterPassword }
      *    - 중요: CI 환경에서는 npx로 설치된 패키지를 사용할 수도 있으므로,
      *      WAIAAS_CLI_PATH 환경변수가 있으면 그것을 사용, 없으면 모노레포 내 경로 사용
      * 5. waitForHealth() 폴링 (최대 15초, 200ms 간격)
      * 6. DaemonInstance 반환
      */
     async start(opts?: { masterPassword?: string; cliPath?: string }): Promise<DaemonInstance>;

     /**
      * 데몬 종료.
      * 1. process.kill('SIGTERM')
      * 2. 5초 대기 후 아직 살아있으면 SIGKILL
      * 3. 임시 디렉토리 삭제 (rmSync recursive)
      */
     async stop(): Promise<void>;

     /** health check 폴링 */
     private async waitForHealth(baseUrl: string, timeoutMs?: number): Promise<void>;

     /** 빈 포트 찾기 (net.createServer 바인드 방식) */
     private async findFreePort(): Promise<number>;
   }
   ```

   - masterPassword 기본값: `'e2e-test-password-12345'`
   - config.toml 작성 시 hostname은 반드시 `"127.0.0.1"` (localhost가 아님, IPv6 이슈 방지)
   - fork된 프로세스의 stdout/stderr를 수집하여 디버깅에 활용 (하지만 기본적으로 콘솔에 출력하지 않음)
   - 프로세스 종료 시 exit code를 확인하고, 비정상 종료 시 stderr 로그 출력

   **사전 조건**: `pnpm turbo run build` 로 데몬이 빌드되어 있어야 함. 테스트 시작 시 packages/cli/bin/waiaas 존재 여부를 확인하고, 없으면 `test.skip`으로 건너뜀.

   ```typescript
   describe('DaemonManager lifecycle', { timeout: 30_000 }, () => {
     let manager: DaemonManager;

     beforeEach(() => { manager = new DaemonManager(); });
     afterEach(async () => { await manager.stop().catch(() => {}); });

     it('starts daemon, passes health check, and stops cleanly', async () => {
       const instance = await manager.start();
       expect(instance.port).toBeGreaterThan(0);
       expect(instance.baseUrl).toContain('http://127.0.0.1');

       // Health check should already pass (start() waits for it)
       const res = await fetch(`${instance.baseUrl}/health`);
       expect(res.status).toBe(200);
       const body = await res.json();
       expect(body.status).toBe('ok');

       // Stop
       await manager.stop();
       // Process should be gone — health check should fail
       await expect(
         fetch(`${instance.baseUrl}/health`).then(r => r.status)
       ).rejects.toThrow(); // ECONNREFUSED
     });
   });
   ```
