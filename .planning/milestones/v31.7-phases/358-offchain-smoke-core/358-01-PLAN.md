SessionManager API:
```typescript
class SessionManager {
  constructor(baseUrl: string, masterPassword: string);
  createWalletAndSession(opts?: { name?: string; chain?: string; environment?: string }): Promise<string>;
  createSession(walletId: string, opts?: { ttl?: number; maxRenewals?: number }): Promise<string>;
  renewSession(): Promise<string>;
  rotateToken(): Promise<string>;
  deleteSession(): Promise<void>;
  get http(): E2EHttpClient;     // sessionAuth client
  get admin(): E2EHttpClient;    // masterAuth client (X-Master-Password)
  get token(): string | undefined;
  get sessionId(): string | undefined;
  get walletId(): string | undefined;
}
async function setupDaemonSession(baseUrl: string, masterPassword?: string): Promise<{ session: SessionManager; http: E2EHttpClient; token: string }>;
```

WAIaaS REST API endpoints (relevant):
- POST /v1/wallets — create wallet (masterAuth, body: { name, chain, environment, createSession? })
- GET /v1/wallets — list wallets (masterAuth)
- GET /v1/wallets/:id — get wallet (masterAuth)
- DELETE /v1/wallets/:id — delete wallet (masterAuth)
- POST /v1/sessions — create session (masterAuth, body: { walletId, ttl?, maxRenewals? })
- GET /v1/sessions — list sessions (masterAuth)
- DELETE /v1/sessions/:id — delete session (masterAuth)
- PUT /v1/sessions/:id/renew — renew session (sessionAuth)
- POST /v1/sessions/:id/rotate — rotate token (masterAuth)
- POST /v1/sessions/:id/wallets — attach wallet to session (masterAuth, body: { walletId })
- DELETE /v1/sessions/:id/wallets/:walletId — detach wallet (masterAuth)
- GET /v1/sessions/:id/wallets — list session wallets (masterAuth)
