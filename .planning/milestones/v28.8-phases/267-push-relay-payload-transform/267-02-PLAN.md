**ntfy-subscriber.ts 변경:**

1. `IPayloadTransformer` import 추가:
```typescript
import type { IPayloadTransformer } from '../transformer/payload-transformer.js';
```

2. `NtfySubscriberOpts`에 optional `transformer` 필드 추가:
```typescript
export interface NtfySubscriberOpts {
  // ... 기존 필드 유지
  transformer?: IPayloadTransformer;
}
```

3. `NtfySubscriber` 클래스에 `private readonly transformer` 필드 추가. constructor에서 `this.transformer = opts.transformer` 저장.

4. `connectSse()` 내부, `buildPushPayload()` 호출 후 transformer 적용:
```typescript
let payload = buildPushPayload(ntfyMsg, type);
if (this.transformer) {
  payload = this.transformer.transform(payload);
}
await this.opts.onMessage(walletName, payload);
```

이렇게 하면:
- transformer가 없으면 (undefined) → 기존 buildPushPayload() 결과 그대로 전달 (bypass)
- transformer가 있으면 → transform()을 거쳐 변환된 payload 전달

**bin.ts 변경:**

1. 상단에 import 추가:
```typescript
import { ConfigurablePayloadTransformer } from './transformer/payload-transformer.js';
import type { IPayloadTransformer } from './transformer/payload-transformer.js';
```

2. provider 초기화 후, transformer 조건부 생성:
```typescript
// Initialize payload transformer (optional)
let transformer: IPayloadTransformer | undefined;
if (config.relay.push.payload) {
  transformer = new ConfigurablePayloadTransformer(config.relay.push.payload);
  console.log('[push-relay] Payload transformer: enabled (static_fields + category_map)');
}
```

3. NtfySubscriber 생성 시 transformer 전달:
```typescript
const subscriber = new NtfySubscriber({
  // ... 기존 필드 유지
  transformer,
});
```

**핵심:** bin.ts에서 `config.relay.push.payload`가 truthy인 경우에만 transformer를 생성하여 전달. undefined이면 NtfySubscriber에 transformer가 전달되지 않아 bypass됨.
