       - `resolveTokenFromAssetId(token: { assetId?: string; address?: string; decimals?: number; symbol?: string }, network: string | undefined, tokenRegistry: TokenRegistryService): Promise<{ token: ResolvedToken; network: string | undefined }>`
       - 로직:
         a. token.assetId가 없으면 `{ token, network }` 그대로 반환

         e. address 채움: token.address가 없으면 parsed.address로 채움. 있으면 case-insensitive 비교 후 불일치 시 에러
         f. 레지스트리 조회: tokenRegistry.getTokensForNetwork(resolvedNetwork)에서 address로 찾기
         g. 레지스트리에 있으면 decimals/symbol 채움 (token에 이미 값이 있으면 유지)
         h. `ResolvedToken` 반환: `{ address, decimals, symbol, assetId }` (decimals/symbol은 undefined 가능)
       - `ResolvedToken` 타입: `{ address: string; decimals?: number; symbol?: string; assetId: string }`
