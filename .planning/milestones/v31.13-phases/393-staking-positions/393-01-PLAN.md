Modify `LidoStakingActionProvider` in `index.ts` to add IPositionProvider methods. The class does NOT formally `implements IPositionProvider` (duck-typing pattern per D5 decision), but adds the three required methods.

**Constructor changes:**
- Accept optional `rpcUrl?: string` in config (or as separate parameter). The provider needs to make `eth_call` to read balances. Store as `this.rpcUrl`.
- Accept optional `priceOracle?: IPriceOracle` for USD conversion.

**Add `getPositions(walletId: string): Promise<PositionUpdate[]>`:**

1. Build two `eth_call` requests to read stETH `balanceOf(walletId)` and wstETH `balanceOf(walletId)`. Use `encodeBalanceOfCalldata()` from lido-contract.ts.
2. For wstETH, also call `stEthPerToken()` to get the exchange rate.
3. Use `fetch()` to make JSON-RPC `eth_call` to `this.rpcUrl` (batch all calls in a single JSON-RPC batch request for efficiency, or sequential -- implementer's discretion).
4. Decode results with `decodeUint256Result()`.
5. For each non-zero balance, create a `PositionUpdate`:
   - `category: 'STAKING'`
   - `provider: 'lido_staking'`
   - `chain: 'ethereum'`
   - `network: 'ethereum-mainnet'`
   - `assetId`: Use `formatCaip19('eip155:1', 'erc20', tokenAddress)` from `@waiaas/core`
   - `amount`: Human-readable string (divide by 1e18). For stETH, raw balance. For wstETH, the wstETH amount itself.
   - `amountUsd`: If priceOracle available, call `getPrice()` for ETH and multiply by stETH-equivalent amount. Otherwise null.
   - `metadata`: `{ token: 'stETH' | 'wstETH', underlyingAmount: string }` where underlyingAmount for wstETH is `(wstETH_balance * stEthPerToken) / 1e18` formatted as string.
   - `status: 'ACTIVE'`
   - `openedAt: Math.floor(Date.now() / 1000)`
6. Wrap entire method in try/catch, return `[]` on any error (per Drift/Kamino pattern).

**Add `getProviderName(): string`:** Return `'lido_staking'`.

**Add `getSupportedCategories(): PositionCategory[]`:** Return `['STAKING']`. Import `PositionCategory` from `@waiaas/core`.

Make a single `eth_call` JSON-RPC request via `fetch()` to `this.rpcUrl`. Return the hex result string. This keeps the main method clean.
