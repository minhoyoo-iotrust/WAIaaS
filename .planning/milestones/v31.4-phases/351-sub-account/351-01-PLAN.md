3. **sub-account-service.ts 생성**: HyperliquidSubAccountService 클래스를 구현한다.
   - Constructor: `(client: HyperliquidExchangeClient, marketData: HyperliquidMarketData, isMainnet: boolean)`
   - `async createSubAccount(name: string, privateKey: Hex): Promise<{ subAccountAddress: string }>`:
     - timestamp = Date.now()
     - hyperliquidChain = isMainnet ? 'Mainnet' : 'Testnet'
     - action = { hyperliquidChain, name, time: timestamp }
     - signature = HyperliquidSigner.signUserSignedAction('CreateSubAccount', action, isMainnet, privateKey)
     - response = client.exchange({ action: { type: 'createSubAccount', name }, nonce: timestamp, signature })
     - 응답에서 sub-account 주소 추출하여 반환
   - `async transfer(params: { subAccountAddress: string, amount: string, isDeposit: boolean, privateKey: Hex }): Promise<ExchangeResponse>`:
     - timestamp = Date.now()
     - usd = Math.round(parseFloat(amount) * 1e6) -- Hyperliquid SubAccountTransfer의 usd는 raw units (1 USDC = 1e6)
     - action = { hyperliquidChain, subAccountUser: subAccountAddress, isDeposit, usd, time: timestamp }
     - signature = HyperliquidSigner.signUserSignedAction('SubAccountTransfer', action, isMainnet, privateKey)
     - response = client.exchange({ action: { type: 'subAccountTransfer', subAccountUser: subAccountAddress, isDeposit, amount: usd }, nonce: timestamp, signature })
   - `async listSubAccounts(walletAddress: Hex): Promise<SubAccountInfo[]>`: marketData.getSubAccounts(walletAddress) 위임
   - `async getSubAccountPositions(walletAddress: Hex, subAccountAddress: Hex): Promise<Position[]>`: marketData.getPositions(walletAddress, subAccountAddress) 위임

4. **index.ts**: HyperliquidSubAccountService + 신규 스키마/타입 re-export 추가

5. **테스트 파일** 작성 (sub-account-service.test.ts):
