    1. Create `packages/daemon/src/services/webhook-delivery-queue.ts`:
       - Export `WebhookDeliveryJob` interface: { webhookId, url, secretEncrypted, eventType, payload (WebhookPayload), deliveryId (UUID v7) }
       - Export `WebhookPayload` interface: { id: string, event: string, timestamp: number, data: Record<string, unknown> }
       - Class `WebhookDeliveryQueue`:
         - Constructor args: `sqlite: SQLiteDatabase, getMasterPassword: () => string`
         - Private constants: maxAttempts=4, timeoutMs=10_000, backoffMs=[0, 1000, 2000, 4000]
         - `enqueue(job: WebhookDeliveryJob): void` -- fire-and-forget: calls `this.deliver(job).catch(console.error)` (no await)
         - `private async deliver(job, attempt=1): Promise<void>`:
           a. Decrypt secret: `decryptSettingValue(job.secretEncrypted, this.getMasterPassword())`
           b. Serialize body: `JSON.stringify(payload)`
           c. Sign: `this.sign(secret, body)`
           d. Build headers: Content-Type, X-WAIaaS-Signature, X-WAIaaS-Event (job.eventType), X-WAIaaS-Delivery (job.deliveryId), X-WAIaaS-Timestamp (Math.floor(Date.now()/1000)), User-Agent: "WAIaaS-Webhook/1.0"
           e. fetch(job.url, { method: 'POST', headers, body, signal: AbortSignal.timeout(this.timeoutMs) })
           f. If 2xx: logAttempt(success, attempt, httpStatus, duration)
           g. If 4xx: logAttempt(failed, attempt, httpStatus, duration) -- NO retry (client error)
           h. If 5xx or network error: logAttempt(failed, attempt, httpStatus/null, duration/null), if attempt < maxAttempts -> setTimeout(backoffMs[attempt], () => deliver(job, attempt+1))
         - `private sign(secret: string, body: string): string`: `sha256=${createHmac('sha256', secret).update(body, 'utf8').digest('hex')}`
         - `private logAttempt(webhookId, eventType, status, httpStatus, attempt, error, requestDuration)`: raw sqlite INSERT into webhook_logs (UUID v7 id, created_at=Math.floor(Date.now()/1000))
