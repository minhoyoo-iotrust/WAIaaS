         d. Sort by created_at descending. Display as table: filename, size (human-readable), created_at, daemon_version, schema_version, files count.

    1. **Create `packages/cli/src/commands/restore.ts`**:
       ```typescript
       export interface RestoreOptions {
         from: string;       // path to .waiaas-backup file
         dataDir?: string;
         password?: string;
         force?: boolean;     // skip confirmation
       }
       ```

       - `restoreCommand(opts: RestoreOptions)`:
         a. **Validate archive exists**: If `opts.from` doesn't exist, error "Backup file not found: {path}" (BACKUP_NOT_FOUND).

         b. **Check daemon not running**: Look for PID file at `join(dataDir, 'daemon.pid')`. If exists, try to check if process is alive (`process.kill(pid, 0)` wrapped in try/catch). If alive, error "WAIaaS daemon is running. Stop it first: waiaas stop" (DAEMON_RUNNING). If PID file exists but process is dead, remove stale PID file and continue.

         c. **Pre-inspect backup**: Call `readArchiveMetadata(readFileSync(opts.from))` to show backup info before proceeding. Display: created_at, daemon_version, schema_version, file count, total size.

         d. **Confirmation prompt**: Unless `--force`, prompt "This will replace your current data with the backup. Continue? (y/N)". If not 'y', exit.

         e. **Resolve password**: `opts.password ?? await resolvePassword()`. Hint: "Enter the master password that was used when creating this backup."

         f. **Decrypt archive**: Read full file buffer. Parse header (readArchiveHeader). Extract encrypted payload = `archive.subarray(60 + metadataLength)`. Derive key from password + salt. Decrypt with AES-256-GCM. Decode entries. If GCM auth fails, error "Wrong master password or corrupted backup" (INVALID_MASTER_PASSWORD).

         g. **Create .bak preservations**:
            - If `join(dataDir, 'data')` dir exists: rename to `data.bak-{timestamp}`
            - If `join(dataDir, 'keystore')` dir exists: rename to `keystore.bak-{timestamp}`
            - If `join(dataDir, 'config.toml')` exists: rename to `config.toml.bak-{timestamp}`
            - Track all renames in `bakPaths` array for rollback.

         h. **Write restored files** (wrapped in try/catch for auto-rollback):
            - `mkdirSync(join(dataDir, 'data'), { recursive: true })`
            - `mkdirSync(join(dataDir, 'keystore'), { recursive: true })`
            - For each entry: if name === 'waiaas.db', write to `join(dataDir, 'data/waiaas.db')`. If name === 'config.toml', write to `join(dataDir, 'config.toml')`. If name starts with 'keystore/', write to `join(dataDir, name)`.

         i. **PRAGMA integrity_check**: Open restored DB with better-sqlite3, run `PRAGMA integrity_check`. If result !== 'ok', throw and trigger rollback.

         j. **Auto-rollback on failure**: If any step g-i fails:
            - Remove partially written files
            - For each item in bakPaths, rename .bak back to original
            - Error "Restore failed: {reason}. Original data has been restored from .bak."

         k. **Success**: Print "Restore complete. Restored from: {filename}". Print "Original data preserved as .bak-{timestamp}". Print "Start the daemon: waiaas start".
