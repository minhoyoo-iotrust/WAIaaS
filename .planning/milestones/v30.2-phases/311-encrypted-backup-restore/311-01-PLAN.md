       - `async createBackup(masterPassword: string): Promise<BackupInfo>`:
         a. `mkdirSync(this.backupsDir, { recursive: true })`
         b. VACUUM INTO: Create temp snapshot path `join(tmpDir, 'waiaas-snap-${Date.now()}.db')`, call `this.sqlite.exec(\`VACUUM INTO '${snapPath}'\`)`, then `readFileSync(snapPath)` to get DB bytes, then `unlinkSync(snapPath)`. tmpDir = `join(this.dataDir, 'tmp')`, ensure exists.
         c. Collect entries: `[{ name: 'waiaas.db', data: dbBuffer }]`. If `config.toml` exists at `join(this.dataDir, 'config.toml')`, add entry. Scan `join(this.dataDir, 'keystore')` for `*.json` files, add each as `keystore/<filename>`.
         d. Encode entries: `encodeEntries(entries)` to get plaintext payload Buffer.
         e. Encrypt: Generate salt (16B randomBytes), nonce (12B randomBytes). `deriveKey(masterPassword, salt)` -> key. `createCipheriv('aes-256-gcm', key, nonce)`, encrypt payload, get authTag. Zero key (`key.fill(0)`).
         f. Build metadata: Get daemon version from `createRequire(import.meta.url)('../../../package.json')` or hardcoded lookup. Get schema_version from `this.sqlite.pragma('user_version')` as number. Compute SHA-256 checksum of encrypted payload. Build BackupMetadata object.
         g. Write archive: `writeArchive({ metadata, salt, nonce, authTag, encryptedPayload })` -> Buffer.
         h. Write to file: filename = `backup-${timestamp}.waiaas-backup` (timestamp = `YYYYMMDD-HHmmss`). `writeFileSync(join(this.backupsDir, filename), archiveBuffer)`.
         i. Return BackupInfo with path, filename, size, created_at, daemon_version, schema_version, file_count.
       - `listBackups(): BackupInfo[]`: Scan backupsDir for `*.waiaas-backup` files. For each, read first 60B + metadata (readArchiveMetadata on the full file buffer or just read enough bytes). Sort by created_at descending. Return BackupInfo[].
       - `inspectBackup(archivePath: string): BackupMetadata`: Read file, call readArchiveMetadata. Throw BACKUP_NOT_FOUND if file doesn't exist.
       - `pruneBackups(keep?: number): number`: List backups sorted newest-first. Delete files beyond `keep` (default 7). Return count deleted.
       - `decryptBackup(archivePath: string, masterPassword: string): Array<{ name: string; data: Buffer }>`: Read file, parse header, extract encrypted payload (after header + metadata). Derive key from password + salt. Decrypt with AES-256-GCM using nonce + authTag. Decode entries. This method is exported for the restore CLI to use.
