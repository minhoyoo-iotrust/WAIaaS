## Milestone Summary

**Key Decisions:**
- IYieldProvider extends IActionProvider (ILendingProvider와 별도, 만기 개념이 고유하기 때문)
- Yield 포지션은 metadata JSON 컬럼 활용 (DDL 변경 없음, Aave/Kamino와 동일 패턴)
- Pendle REST API v2 Convert 엔드포인트 사용 (SDK 의존성 불필요, DEC-1)
- MaturityMonitor 1일 1회 폴링 (만기는 초 단위 추적 불필요, DEC-3)

**Issues Resolved:**
- #216: Solana WSS URL prefix (`wss://` → `wss://api.` 변환 수정)
- #217: Lido factory default network residue (factory 함수 시그니처 수정)

**Issues Deferred:**
- #164: IncomingTxMonitorService 환경 기본 네트워크만 구독 (MEDIUM)
- STO-03: Confirmation Worker RPC 콜백 미주입

**Technical Debt Incurred:**
- Pendle 테스트는 mock 기반 (실제 Pendle API 호출 E2E 테스트 없음)
- PT/YT 가격 오라클 미통합 (IPriceOracle에서 PT/YT 토큰 가격 미지원)
