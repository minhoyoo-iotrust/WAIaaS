  <behavior>
    - ensureApiKeys(walletId, walletAddress, privateKey) returns { apiKey, secret, passphrase }
    - If api_keys exist in DB for wallet, decrypt and return them
    - If no api_keys, sign ClobAuth EIP-712 -> POST /auth/api-key -> encrypt secret+passphrase -> INSERT DB -> return
    - getApiKeys(walletId) returns decrypted credentials or null
    - deleteApiKeys(walletId, walletAddress, privateKey) calls DELETE /auth/api-key + removes DB row
    - Encrypted fields use same encryption as existing wallet private key storage (sodium secretbox with master password derived key)
  </behavior>
  <action>
    1. Create `api-key-service.ts`: PolymarketApiKeyService class.
       Constructor: (clobClient: PolymarketClobClient, db: DB, encryptFn, decryptFn) where encryptFn/decryptFn are the existing master-password-based encryption functions from the daemon's crypto module.

       Methods:
       - `ensureApiKeys(walletId, walletAddress, privateKey)`: Check polymarket_api_keys for wallet_id. If found, decrypt api_secret_encrypted and api_passphrase_encrypted, return { apiKey, secret, passphrase }. If not found:
         a. Generate timestamp = String(Math.floor(Date.now() / 1000))
         b. Sign ClobAuth via PolymarketSigner.signClobAuth(walletAddress, timestamp, 0n, privateKey)
         c. Call clobClient.createApiKey(walletAddress, signature, timestamp)
         d. Encrypt secret and passphrase with encryptFn
         e. INSERT into polymarket_api_keys (id=uuidv7, wallet_id, api_key, api_secret_encrypted, api_passphrase_encrypted, signature_type=0)
         f. Return { apiKey, secret, passphrase }

       - `getApiKeys(walletId)`: SELECT from polymarket_api_keys, decrypt, return or null.

       - `deleteApiKeys(walletId, walletAddress, privateKey)`: Sign ClobAuth, call clobClient.deleteApiKey, DELETE from polymarket_api_keys.

    2. Write tests: Mock clobClient, encryptFn/decryptFn, and DB. Test lazy creation path (no existing keys), cache hit path (keys exist), deletion, and encryption round-trip.
  </action>
