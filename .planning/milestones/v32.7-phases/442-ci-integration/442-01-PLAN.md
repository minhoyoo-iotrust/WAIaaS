2. **Node.js + pnpm 빌드 스텝 추가 (CI-01)**:
   checkout 다음, Upload artifact 앞에 아래 스텝 삽입:

   ```yaml
   - name: Setup Node.js
     uses: actions/setup-node@v4
     with:
       node-version: 22

   - name: Install pnpm
     uses: pnpm/action-setup@v4

   - name: Install build dependencies
     run: pnpm install --frozen-lockfile --filter .
     # 루트 devDependencies만 설치 (gray-matter, marked, highlight.js 등)

   - name: Build site
     run: node site/build.mjs
   ```

   `--filter .` 를 사용하여 루트 패키지만 설치. 빌드에 필요한 gray-matter, marked, marked-highlight, highlight.js가 모두 루트 devDependencies에 있음.

3. **Upload artifact path 유지**: `path: site` (빌드 결과가 site/ 하위에 생성되므로 동일)

최종 스텝 순서: checkout → setup-node → pnpm → install → build → configure-pages → upload-artifact → deploy
