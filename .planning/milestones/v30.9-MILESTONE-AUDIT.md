## 5. Gaps and Remediation

### GAP-01 [HIGH]: Stale Skill Files

**Files affected:**
- `/Users/minho.yoo/dev/wallet/WAIaaS/skills/admin.skill.md` (lines 462-467, 487, 609-626)
- `/Users/minho.yoo/dev/wallet/WAIaaS/skills/wallet.skill.md` (lines 56, 81, 125-127)

**Problem:** Skill files still document the removed global settings model. `admin.skill.md` shows `smart_account.bundler_url`, `smart_account.paymaster_url`, `smart_account.paymaster_api_key` as valid settings (all 23 removed in PROV-09). `wallet.skill.md` instructs agents that `smart_account.enabled` defaults to `false` (changed to `true` in DFLT-01) and requires `smart_account.bundler_url` (removed). The new `PUT /v1/wallets/:id/provider` endpoint is not documented in any skill file.

**Risk:** AI agents reading these skill files will attempt to use removed API patterns, causing 400 errors.

**Fix:** Update both skill files to reflect the per-wallet provider model: remove references to deleted settings keys, document `PUT /v1/wallets/:id/provider`, update Smart Account creation instructions.

**Fix:** Update checkboxes to `[x]` and traceability status to "Complete" for all 7 requirements.

### GAP-03 [MEDIUM]: Objective File Not Updated

**File:** `/Users/minho.yoo/dev/wallet/WAIaaS/internal/objectives/m30-09-smart-account-dx.md`

**Problem:** Status header still shows `PLANNED`. Per CLAUDE.md: "When a milestone is shipped, update the corresponding objective status header: PLANNED -> SHIPPED, add Completed: YYYY-MM-DD."

### GAP-04 [MEDIUM]: Missing VERIFICATION.md Files

**Problem:** No VERIFICATION.md files exist for Phases 324, 325, or 326 despite `config.json` having `verifier: true`.

**Fix:** Create phase verification files or document that verification was performed via this milestone audit.

**Problem:** `@waiaas/sdk` has no wrapper for `PUT /v1/wallets/:id/provider` or provider status query. Agents using the SDK must use raw HTTP calls for provider management.

**Fix:** Consider adding `setProvider(walletId, config)` to SDK in a future milestone.
