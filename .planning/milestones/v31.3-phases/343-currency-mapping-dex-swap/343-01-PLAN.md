    **2. Create `currency-mapper.ts`** at `packages/actions/src/providers/dcent-swap/currency-mapper.ts`:
    - Import `parseCaip19`, `formatCaip19` from `@waiaas/core`
    - Import `parseCaip2` from `@waiaas/core`
    - Define `NATIVE_DCENT_ID_MAP`: EVM chainId number -> DCent name. Map: { 1: 'ETHEREUM', 56: 'BSC', 137: 'POLYGON', 8217: 'KLAYTN', 50: 'XINFIN' }
    - Define reverse map `DCENT_NATIVE_TO_CAIP2`: DCent name -> { caip2, slip44 }. Map: { ETHEREUM: { caip2: 'eip155:1', slip44: 60 }, BSC: { caip2: 'eip155:56', slip44: 60 }, POLYGON: { caip2: 'eip155:137', slip44: 966 }, KLAYTN: { caip2: 'eip155:8217', slip44: 60 }, XINFIN: { caip2: 'eip155:50', slip44: 60 }, SOLANA: { caip2: 'solana:5eykt4UsFv8P8NJdTREpY1vzqKqZKvdp', slip44: 501 } }
    - Define `EVM_TOKEN_DEVICE_MAP`: chainId number -> tokenDeviceId prefix. Map: { 1: 'ERC20', 56: 'BEP20', 137: 'POLYGON-ERC20', 8217: 'KLAYTN-ERC20' }. Default pattern: `CH20:{chainId}`
    - Define reverse `TOKEN_PREFIX_TO_CHAIN`: { ERC20: 1, BEP20: 56, 'POLYGON-ERC20': 137, 'KLAYTN-ERC20': 8217 }
    - `caip19ToDcentId(caip19: string): string`:
      1. Parse CAIP-19 -> { chainId, assetNamespace, assetReference }
      2. Parse CAIP-2 -> { namespace, reference }
      3. If namespace === 'solana':
         - If assetNamespace === 'slip44' -> 'SOLANA'
         - If assetNamespace === 'token' -> `SPL-TOKEN/${assetReference}`
      4. If namespace === 'eip155':
         - chainIdNum = parseInt(reference)
         - If assetNamespace === 'slip44' (native):
           - Check NATIVE_DCENT_ID_MAP[chainIdNum] -> return name or `CHAN:{chainIdNum}`
         - If assetNamespace === 'erc20' (token):
           - Check EVM_TOKEN_DEVICE_MAP[chainIdNum] -> return `{prefix}/{assetReference}` or `CH20:{chainIdNum}/{assetReference}`
      5. Otherwise throw ChainError('UNSUPPORTED_ASSET', ...)
    - `dcentIdToCaip19(dcentId: string): string`:
      1. Check DCENT_NATIVE_TO_CAIP2[dcentId] -> formatCaip19(caip2, 'slip44', slip44)
      2. If starts with 'CHAN:' -> parse chainId, formatCaip19(`eip155:{chainId}`, 'slip44', '60')
      3. If starts with 'SPL-TOKEN/' -> formatCaip19(solana_caip2, 'token', mintAddress)
      4. If starts with 'CH20:' -> parse `CH20:{chainId}/{addr}`, formatCaip19(`eip155:{chainId}`, 'erc20', addr)
      5. Check TOKEN_PREFIX_TO_CHAIN for known prefix (split on '/'): formatCaip19(`eip155:{chainId}`, 'erc20', addr)
      6. Otherwise throw ChainError('UNSUPPORTED_ASSET', ...)
    - IMPORTANT: Polygon slip44 is 966 (per codebase NATIVE_SLIP44), NOT 60 as the design doc section 5.3 incorrectly states. Use 966 for Polygon.
    - IMPORTANT: Solana SPL tokens use `token:` namespace (per codebase tokenAssetId), NOT `spl:` as the design doc section 5.3 states. Use `token:`.
