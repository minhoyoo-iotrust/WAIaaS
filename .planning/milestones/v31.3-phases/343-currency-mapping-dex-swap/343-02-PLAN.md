// encodeApproveCalldata pattern:
function encodeApproveCalldata(spender: string, amount: string): string {
  const selector = '0x095ea7b3';
  const paddedSpender = spender.slice(2).toLowerCase().padStart(64, '0');
  const paddedAmount = BigInt(amount).toString(16).padStart(64, '0');
  return `${selector}${paddedSpender}${paddedAmount}`;
}
// ERC-20 sell: [approveRequest, swapRequest]
// Native sell: [swapRequest]

    **1. Create `dex-swap.ts`** at `packages/actions/src/providers/dcent-swap/dex-swap.ts`:

    - Import caip19ToDcentId, dcentIdToCaip19 from currency-mapper
    - Import DcentSwapApiClient, schemas, config

    - `encodeApproveCalldata(spender: string, amount: string): string` — same pattern as zerox-swap (selector 0x095ea7b3, padded spender + amount). DS-02: self-encode because DCent approve API is partially unimplemented.

    - `async getDcentQuotes(client: DcentSwapApiClient, params: { fromAsset: string, toAsset: string, amount: string, fromDecimals: number, toDecimals: number }): Promise<DcentQuoteResult>`:
      1. Convert fromAsset/toAsset CAIP-19 -> DCent IDs via caip19ToDcentId
      2. Call client.getQuotes({ fromId, toId, amount, fromDecimals, toDecimals })
      3. If response.status === 'fail_no_available_provider', throw ChainError('NO_ROUTE')
      4. Filter providers.common to status === 'success' only
      5. Separate into dexProviders (providerType 'swap' or 'cross_swap') and exchangeProviders (providerType 'exchange')
      6. Sort dexProviders by expectedAmount descending (BigInt comparison)
      7. Return { dexProviders, exchangeProviders, bestDexProvider: dexProviders[0] || null, bestOrder: response.providers.bestOrder, rawResponse: response }
      8. Define DcentQuoteResult type with these fields

    - `async executeDexSwap(client: DcentSwapApiClient, params: { fromAsset: string, toAsset: string, amount: string, fromDecimals: number, toDecimals: number, walletAddress: string, providerId?: string, slippageBps?: number }, config: DcentSwapConfig): Promise<ContractCallRequest[]>`:
      1. Block same-asset swap (fromAsset === toAsset -> INVALID_INSTRUCTION)
      2. Get quotes via getDcentQuotes
      3. If no dexProviders available, throw ChainError('NO_ROUTE')
      4. Select provider: params.providerId ?? first DEX provider from bestOrder (filter by providerType !== 'exchange')
      5. Validate selected provider has status 'success'
      6. Check expectedAmount: if '0' throw ChainError('AMOUNT_TOO_SMALL')
      7. Clamp slippage: clampSlippageBps(params.slippageBps ?? 0, asBps(config.defaultSlippageBps), asBps(config.maxSlippageBps))
      8. Convert slippage from bps to integer percent for DCent API: Math.round(slippageBps / 100)
      9. Call client.getDexSwapTransactionData({ fromId, toId, fromAmount: params.amount, fromDecimals, toDecimals, fromWalletAddress: walletAddress, toWalletAddress: walletAddress, providerId: selectedProvider.providerId, isAutoSlippage: false, slippage })
      10. Build ContractCallRequest[]:
          - Determine if native sell: check fromAsset CAIP-19 assetNamespace === 'slip44'
          - If ERC-20 sell: extract token contract address from fromAsset CAIP-19 (assetReference), build approve request using encodeApproveCalldata(spenderContractAddress, amount)
          - Build swap request: { type: 'CONTRACT_CALL', to: txdata.to, calldata: txdata.data, value: txdata.value ? BigInt(txdata.value).toString() : '0' }
          - Return [approveRequest?, swapRequest]

    - Test executeDexSwap approve calldata: correct selector + padded spender + amount
