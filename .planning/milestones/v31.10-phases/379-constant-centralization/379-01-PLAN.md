**1. `packages/daemon/src/constants.ts`:**
```typescript
/**
 * Daemon package shared constants.
 * Extracted from repeated magic numbers across daemon services.
 */

/** Oracle HTTP request timeout (coingecko, pyth). */
export const ORACLE_TIMEOUT_MS = 5_000;

/** Signing channel (ntfy, wallet-notification) HTTP fetch timeout. */
export const SIGNING_CHANNEL_FETCH_TIMEOUT_MS = 10_000;

/** Default max retries for HTTP clients (nft-indexer, etc). */
export const DEFAULT_MAX_RETRIES = 3;

/** Gas safety margin numerator (120% = 120n/100n). Per CLAUDE.md mandate. */
export const GAS_SAFETY_NUMERATOR = 120n;

/** Gas safety margin denominator. */
export const GAS_SAFETY_DENOMINATOR = 100n;

/** Worker: graceful shutdown deadline (ms). */
export const WORKER_SHUTDOWN_DEADLINE_MS = 5_000;
```

**2. `packages/cli/src/constants.ts`:**
```typescript
/**
 * CLI package shared constants.
 */

/** Default daemon port for CLI commands. */
export const DEFAULT_DAEMON_PORT = 3100;

/** HTTP fetch timeout for CLI health checks and API calls. */
export const CLI_FETCH_TIMEOUT_MS = 5_000;
```

**3. `packages/admin/src/constants.ts`:**
```typescript
/**
 * Admin UI shared constants.
 */

/** Hyperliquid component polling interval (SubAccountDetail, PositionsTable, etc). */
export const HYPERLIQUID_POLL_INTERVAL_MS = 10_000;

/** Dashboard/notification log polling interval. */
export const DASHBOARD_POLL_INTERVAL_MS = 30_000;
```

주의사항:
- 단일 파일에서만 사용되고 이미 명명된 상수(e.g., `CACHE_TTL_SECONDS = 86400` in nft-metadata-cache.ts)는 그대로 둔다.
- Admin Settings에서 이미 관리되는 값(e.g., `autostop.idle_check_interval_sec`)은 추출하지 않는다.
- config.toml에서 읽어오는 값(e.g., `config.database.wal_checkpoint_interval`)은 추출하지 않는다.
- `Math.floor(Date.now() / 1000)` 같은 시간 변환 패턴은 매직 넘버가 아니므로 건드리지 않는다.
