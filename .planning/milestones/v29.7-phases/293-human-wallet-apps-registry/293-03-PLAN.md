**Header section:**
- Page title "Human Wallet Apps" (handled by layout.tsx PAGE_TITLES)

**App cards section:**
- Fetch wallet apps from GET /v1/admin/wallet-apps on mount
- For each app, render a card (use existing card styling pattern from other pages):
  - **Display name** as card header
  - **App name** in smaller text below (e.g., `dcent`)
  - **Signing toggle**: checkbox/toggle for signing_enabled. On change: PUT /v1/admin/wallet-apps/:id { signing_enabled: !current }
  - **Alerts toggle**: checkbox/toggle for alerts_enabled. On change: PUT /v1/admin/wallet-apps/:id { alerts_enabled: !current }
  - **Used by**: list of wallet labels (from used_by field). If empty, show "No wallets"
  - **Remove button**: red button at bottom-right. On click: confirm dialog, then DELETE /v1/admin/wallet-apps/:id

**Register App section:**
- [+ Register App] button at the bottom
- On click: open a Modal with name + display_name fields
- On submit: POST /v1/admin/wallet-apps with { name, display_name }
- On 409: show toast "App already registered"
- On success: refresh app list, close modal
