    **1. Create tray icon PNGs (6 files):**
    Generate minimal colored circle PNG icons programmatically using a Rust build script or create them manually:
    - tray-green.png: 32x32 green filled circle on transparent background
    - tray-yellow.png: 32x32 yellow/orange filled circle on transparent background
    - tray-red.png: 32x32 red filled circle on transparent background
    - @2x variants: 64x64 versions of each for Retina displays

    Use a simple approach: create the PNGs with a script (e.g., Python with Pillow, or ImageMagick `convert` if available). If neither available, create placeholder PNGs with `printf` binary data for minimal valid PNGs. The icons should be simple colored dots -- these are system tray status indicators.

    **2. Create apps/desktop/src-tauri/src/tray.rs** (per design doc section 5):

    ```rust
    // Core structure:
    use tauri::{
        AppHandle, Manager,
        tray::{TrayIconBuilder, TrayIconEvent, MouseButton, MouseButtonState},
        menu::{Menu, MenuItem, PredefinedMenuItem},
        image::Image,
    };
    use crate::sidecar::SidecarManager;
    use crate::types::{DaemonState, HealthStatus};
    ```

    **TrayIcon enum** with Green, Yellow, Red variants.

    **determine_tray_icon(status: &DaemonStatus) -> TrayIcon** function (per design doc 5.2):
    - Not running / Crashed / Error state -> Red
    - HealthStatus::Unhealthy -> Red
    - restart_count > 0 -> Yellow
    - HealthStatus::Unknown -> Yellow
    - Otherwise -> Green

    **setup_tray(app: &tauri::App) -> Result<(), Box<dyn std::error::Error>>** function:
    - Build context menu with MenuItems: "WAIaaS Desktop" (disabled title), separator, "Open Dashboard", separator, "Start Daemon", "Stop Daemon", "Restart Daemon", separator, "Quit WAIaaS"
    - Create TrayIconBuilder with initial green icon, tooltip "WAIaaS Desktop", menu
    - Register menu event handler:
      - "open_dashboard": get main window, show + set_focus
      - "start_daemon": spawn async, call manager.start()
      - "stop_daemon": spawn async, call manager.stop(false)
      - "restart_daemon": spawn async, call manager.stop(false), sleep 1s, manager.start()
      - "quit_waiaas": spawn async, call manager.stop(false), then app.exit(0)
    - Register tray icon click event: on left click, show + focus main window

    **start_tray_polling(app_handle: AppHandle)** function:
    - Spawn tokio task with 30-second interval loop
    - Each tick: get SidecarManager state, call check_health(), determine_tray_icon()
    - Update tray icon based on color (load appropriate PNG from icons/)
    - Update menu items: disable "Start Daemon" if running, disable "Stop Daemon" if not running
    - Use `app_handle.tray_by_id("main")` to get tray reference

    **Icon loading**: Use `Image::from_bytes()` with `include_bytes!()` macro pointing to the icon PNGs. Store in a lazy static or load on each update (tray updates are infrequent at 30s intervals, so loading is fine).

    Note: Tauri 2.x tray API uses `tray::TrayIconBuilder` (not `SystemTray` from 1.x). Menu uses `menu::Menu::with_items()`. Check Tauri 2 docs for exact API.
