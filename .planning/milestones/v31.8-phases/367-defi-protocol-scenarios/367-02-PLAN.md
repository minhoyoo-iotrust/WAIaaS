- frontmatter: id=defi-02, category=defi, network=[ethereum-mainnet, polygon-mainnet], requires_funds=true, estimated_cost_usd=5.00, risk_level=medium, tags=[defi, swap, 0x, evm, dex]
- Prerequisites: 데몬 실행, 세션 토큰, Ethereum Mainnet 또는 Polygon 지갑, ETH 보유 (스왑 + 가스비)
- Steps:
  (1) ETH/토큰 잔액 조회 (network=ethereum-mainnet 또는 polygon-mainnet)
  (2) 0x 스왑 견적 dry-run (POST /v1/transactions/dry-run, type=CONTRACT_CALL, action=0x-swap, params: {sellToken: "ETH" 또는 "0xEeeeeE...", buyToken: "0xA0b869..." (USDC), sellAmount: "0.001"}, network=ethereum-mainnet) — 예상 수령량, 가스비 확인

  (4) 실제 스왑 실행 (POST /v1/transactions, 동일 params)
  (5) tx 상태 확인 — confirmed 대기
  (6) 잔액 재확인 — ETH 감소, USDC 증가 확인
- Estimated Cost: Ethereum ~$3-5 (approve + swap gas), Polygon ~$0.05
- Troubleshooting: insufficient ETH for gas, 0x API rate limit, price impact too high, USDC approve needed (첫 스왑)
- 참고: Polygon에서 실행 시 가스비가 크게 절감됨. 사용자에게 Polygon 우선 안내 가능

- frontmatter: id=defi-07, category=defi, network=[ethereum-mainnet, polygon-mainnet], requires_funds=true, estimated_cost_usd=5.00, risk_level=medium, tags=[defi, lending, aave, evm, supply]
- Prerequisites: EVM 지갑, ETH/POL (가스비) + USDC 보유 (최소 1 USDC). Polygon에서 실행 시 가스비 절감.
- Steps:
  (1) USDC 토큰 잔액 조회
  (2) Aave V3 마켓 정보 확인 — USDC supply APY (connect-info 또는 관련 API)
  (3) USDC approve dry-run (type=APPROVE, spender=Aave Pool address, amount) — 첫 supply 시 필요
  (4) 사용자 승인 후 실제 approve 실행 (필요한 경우)
  (5) Dry-run USDC supply (POST /v1/transactions/dry-run, type=CONTRACT_CALL, action=aave-supply, params: {asset: "USDC", amount: "1.0"}, network=ethereum-mainnet 또는 polygon-mainnet)
  (6) 사용자 승인 후 실제 supply 실행
  (7) tx 확인
  (8) Aave 포지션 확인 — supply 포지션이 생성되었는지 확인 (DeFi positions API)
  (9) (선택) Withdraw dry-run 확인 (action=aave-withdraw) — 실제 실행은 사용자 선택
- Estimated Cost: Ethereum ~$5 (approve + supply), Polygon ~$0.05
- Troubleshooting: USDC approve 필요, insufficient USDC, Aave pool paused, aToken 잔액 미반영 (인덱싱 지연)
- 참고: Polygon에서 실행을 권장하여 가스비 절감. USDC 주소는 네트워크별 다름 (Ethereum: 0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48, Polygon: 0x3c499c542cEF5E3811e1192ce70d8cC03d5c3359)
