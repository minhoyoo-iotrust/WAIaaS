**lifi-bridge.md** (defi-03, LI.FI Cross-Chain Bridge L1 -> L2):
- frontmatter: id=defi-03, category=defi, network=[ethereum-mainnet, arbitrum-mainnet], requires_funds=true, estimated_cost_usd=5.00, risk_level=medium, tags=[defi, bridge, lifi, crosschain, l1-to-l2]
- Prerequisites: 데몬 실행, 세션 토큰, Ethereum Mainnet EVM 지갑, ETH 보유 (브릿지 금액 + 가스비, 최소 0.01 ETH). 수신 지갑: Arbitrum Mainnet 지갑 (동일 주소 권장)
- Steps:
  (1) 출발 체인 ETH 잔액 조회 (network=ethereum-mainnet)
  (2) LI.FI 브릿지 견적 dry-run (POST /v1/transactions/dry-run, type=CONTRACT_CALL, action=lifi-bridge, params: {fromChain: "ethereum-mainnet", toChain: "arbitrum-mainnet", fromToken: "ETH", toToken: "ETH", amount: "0.005", toAddress: "<MY_ARBITRUM_ADDRESS>"}, network=ethereum-mainnet) — 예상 수령량, 브릿지 경로, 가스비 확인
  (3) 사용자에게 브릿지 경로, 예상 수령량, 소요 시간, 가스비 표시, 승인 요청
  (4) 실제 브릿지 실행
  (5) 출발 체인 tx 확인 — confirmed
  (6) 브릿지 상태 추적 — LI.FI status API 또는 tx status로 진행 상황 확인 (PENDING -> SENDING -> DONE). 일반적으로 2-15분 소요
  (7) 도착 체인 잔액 확인 (network=arbitrum-mainnet) — ETH 잔액 증가 확인
- Estimated Cost: L1 gas ~$3-5 + bridge fee (~0.1-0.5%)
- Troubleshooting: bridge route not available, gas price spike on L1, bridge timeout (10분 이상 대기), destination token different from expected
- 중요: 브릿지 완료까지 시간이 걸리므로 사용자에게 대기 안내. Status polling 간격 30초 권장

**across-bridge.md** (defi-04, Across Cross-Chain Bridge L2 -> L2):
- frontmatter: id=defi-04, category=defi, network=[arbitrum-mainnet, base-mainnet], requires_funds=true, estimated_cost_usd=0.50, risk_level=medium, tags=[defi, bridge, across, crosschain, l2-to-l2]
- Prerequisites: Arbitrum Mainnet EVM 지갑, USDC 또는 ETH 보유 (최소 1 USDC 또는 0.001 ETH). 수신: Base Mainnet 지갑
- Steps:
  (1) 출발 체인(Arbitrum) 잔액 조회
  (2) Across 브릿지 견적 dry-run (POST /v1/transactions/dry-run, type=CONTRACT_CALL, action=across-bridge, params: {fromChain: "arbitrum-mainnet", toChain: "base-mainnet", token: "USDC", amount: "1.0", toAddress: "<MY_BASE_ADDRESS>"}, network=arbitrum-mainnet) — SpokePool depositV3 예상
  (3) 사용자에게 수수료, 예상 수령량, 소요 시간 표시, 승인 요청
  (4) USDC approve dry-run + 실행 (SpokePool에 대한 approve, 필요한 경우)
  (5) 실제 브릿지 실행
  (6) 출발 체인 tx 확인
  (7) Across 브릿지 상태 추적 — 2-phase polling (deposit confirmed -> fill completed). Across는 ~2-10분 소요
  (8) 도착 체인(Base) 잔액 확인 — USDC 잔액 증가
- Estimated Cost: Arbitrum gas ~$0.05 + Across relayer fee (~0.05-0.1%)
- Troubleshooting: SpokePool address mismatch, approve needed, bridge fill delayed, relayer fee spike
