**hyperliquid-spot-perp.md** (testnet-06, Hyperliquid Testnet):
- frontmatter: id=testnet-06, category=testnet, network=[hyperliquid-testnet], requires_funds=true, estimated_cost_usd=0, risk_level=medium, tags=[hyperliquid, spot, perp, order, testnet]
- Prerequisites: 데몬 실행, 세션 토큰, Hyperliquid Testnet 연결 설정 (Admin Settings에서 hyperliquid_testnet=true), Hyperliquid testnet 자금 (https://app.hyperliquid-testnet.xyz/drip)
- Steps:
  (1) Hyperliquid 잔액 확인 — GET /v1/wallets/{id}/balance?network=hyperliquid-testnet
  (2) Spot 주문 생성 — POST /v1/transactions (type=CONTRACT_CALL, action=hyperliquid-spot-order, params={coin: "ETH", size: "0.01", price: "1000", side: "buy", orderType: "limit"})
  (3) Spot 주문 확인 — 주문 ID 기록, GET /v1/wallets/{id}/hyperliquid/orders?type=spot
  (4) Spot 주문 취소 — POST /v1/transactions (action=hyperliquid-spot-cancel, params={coin: "ETH", orderId: ...})
  (5) Perp 주문 생성 — POST /v1/transactions (action=hyperliquid-perp-order, params={coin: "ETH", size: "0.01", price: "1000", side: "buy", orderType: "limit", leverage: 2})
  (6) Perp 주문 확인 및 취소
  (7) 최종 잔액 확인 — 주문 취소했으므로 잔액 변동 최소
- Estimated Cost: $0 (Hyperliquid testnet 무료)
- Troubleshooting: testnet 설정 누락, drip faucet 미사용, rate limit
