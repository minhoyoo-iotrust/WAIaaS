**eth-transfer.md** (mainnet-01, Ethereum Mainnet ETH 전송):
- frontmatter: id=mainnet-01, category=mainnet, network=[ethereum-mainnet], requires_funds=true, estimated_cost_usd=0.50, risk_level=medium, tags=[transfer, eth, native, mainnet]
- Prerequisites: 데몬 실행, 세션 토큰, Ethereum Mainnet EVM 지갑, ETH 보유 (가스비 포함 최소 0.002 ETH)
- Steps: (1) 잔액 조회 (2) dry-run 자기 전송 (type=TRANSFER, to=자기주소, value=0.001 ETH, network=ethereum-mainnet) — 가스비 확인 후 사용자 승인 (3) 실제 전송 (4) tx 상태 확인 (5) 잔액 재확인 — 가스비만큼만 감소
- Estimated Cost: ~21,000 gas, ~$0.50 (mainnet gas price에 따라 변동)
- Troubleshooting: gas price 급등, insufficient ETH, pending tx

**sol-transfer.md** (mainnet-02, Solana Mainnet SOL 전송):
- frontmatter: id=mainnet-02, category=mainnet, network=[solana-mainnet], requires_funds=true, estimated_cost_usd=0.001, risk_level=medium, tags=[transfer, sol, native, mainnet]
- Prerequisites: Solana Mainnet 지갑, SOL 보유 (최소 0.01 SOL)
- Steps: 잔액 조회 → dry-run → 자기 전송 0.001 SOL → tx 확인 → 잔액 재확인
- Estimated Cost: ~$0.001 (Solana tx fee ~0.000005 SOL)

**erc20-transfer.md** (mainnet-03, Mainnet ERC-20 USDC 전송):
- frontmatter: id=mainnet-03, category=mainnet, network=[ethereum-mainnet], requires_funds=true, estimated_cost_usd=1.00, risk_level=medium, tags=[transfer, erc20, usdc, token, mainnet]
- Prerequisites: ETH (가스비) + USDC 보유. USDC contract: 0xA0b86991c6218b36c1d19D4a2e9Eb0cE3606eB48
- Steps: (1) 토큰 잔액 조회 (2) dry-run 자기 전송 (type=TOKEN_TRANSFER, token=USDC주소, amount=0.01, to=자기주소) — ERC-20 transfer는 approve 불필요 (자기 전송) (3) 실제 전송 (4) tx 확인 (5) 토큰 잔액 불변 확인 (자기 전송), ETH 가스비만 감소
- Estimated Cost: ~65,000 gas, ~$1.00

**spl-transfer.md** (mainnet-04, Mainnet SPL USDC 전송):
- frontmatter: id=mainnet-04, category=mainnet, network=[solana-mainnet], requires_funds=true, estimated_cost_usd=0.001, risk_level=medium, tags=[transfer, spl, usdc, token, mainnet]
- Prerequisites: SOL (tx fee) + USDC SPL 보유. USDC mint: EPjFWdd5AufqSSqeM2qN1xzybapC8G4wEGGkZwyTDt1v
- Steps: 토큰 잔액 조회 → dry-run → 자기 전송 0.01 USDC → tx 확인 → 잔액 재확인
- Estimated Cost: ~$0.001

**l2-native-transfer.md** (mainnet-05, L2 네이티브 전송):
- frontmatter: id=mainnet-05, category=mainnet, network=[polygon-mainnet, arbitrum-mainnet, base-mainnet], requires_funds=true, estimated_cost_usd=0.05, risk_level=medium, tags=[transfer, l2, polygon, arbitrum, base, native, mainnet]
- Prerequisites: L2 네트워크 지갑, 각 L2 네이티브 토큰 보유 (POL/ETH/ETH)
- Steps: 3개 L2를 순차적으로 테스트 — 각각 잔액 조회 → dry-run → 자기 전송 → tx 확인 → 잔액 확인. 사용자가 보유한 L2 중 선택 가능하도록 안내.
  (1) 사용 가능한 L2 지갑 확인 (connect-info에서 네트워크별 지갑)
  (2) Polygon 자기 전송 (network=polygon-mainnet, value=0.001 POL) 또는 사용자 선택 L2
  (3) Arbitrum 자기 전송 (network=arbitrum-mainnet, value=0.0001 ETH) — 선택적
  (4) Base 자기 전송 (network=base-mainnet, value=0.0001 ETH) — 선택적
  (5) 각 L2 잔액 재확인
- Estimated Cost: ~$0.01-0.05 per L2 (L2 가스비는 L1보다 훨씬 저렴)

**nft-transfer.md** (mainnet-06, Mainnet NFT 전송):
- frontmatter: id=mainnet-06, category=mainnet, network=[ethereum-mainnet], requires_funds=true, estimated_cost_usd=2.00, risk_level=medium, tags=[nft, erc721, erc1155, transfer, mainnet]
- Prerequisites: ETH (가스비), 보유 NFT 1개 이상 (ERC-721 또는 ERC-1155). NFT 미보유 시 시나리오 스킵 안내.
- Steps: (1) NFT 목록 조회 (2) 보유 NFT 중 하나 선택 (사용자 확인) (3) dry-run 자기 전송 (type=NFT_TRANSFER) — 가스비 확인 (4) 실제 자기 전송 (5) tx 확인 (6) NFT 여전히 소유 확인
- Estimated Cost: ~$2.00 (ERC-721 transfer ~50,000 gas at mainnet price)
- 주의: NFT 미보유 시 SKIP 처리. 강제 실행하지 않음.
