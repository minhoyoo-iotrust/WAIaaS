  <behavior>
    ## ISignerCapability 인터페이스 + SigningParams union

    - ISignerCapability has readonly scheme, canSign(params), sign(params) per doc-81 D2.1
    - SigningParams is discriminated union on scheme field with 7 variants per doc-81 D2.2
    - SigningResult has signature (string | Uint8Array) + optional metadata
    - SigningError extends Error with scheme, code, cause per doc-81 D2.3

    ## 7종 Capability canSign() + sign() 검증

    ### Eip712SignerCapability (scheme: 'eip712')
    - canSign: scheme === 'eip712' && privateKey exists -> true
    - canSign: wrong scheme -> false
    - sign: viem signTypedData with domain/types/primaryType/value -> hex signature string
    - sign: invalid key -> SigningError INVALID_KEY

    ### PersonalSignCapability (scheme: 'personal')
    - canSign: scheme === 'personal' && privateKey exists -> true
    - sign: viem signMessage -> hex signature string

    ### Erc8128SignerCapability (scheme: 'erc8128')
    - canSign: scheme === 'erc8128' && privateKey && chainId && address exist -> true
    - sign: delegates to erc8128Core.signHttpMessage() -> metadata with signatureInput, signature, etc.

    ### HmacSignerCapability (scheme: 'hmac-sha256')
    - canSign: scheme === 'hmac-sha256' && secret exists -> true
    - sign: crypto.createHmac('sha256', secret).update(data).digest(encoding) -> signature string
    - sign: encoding default 'hex', also supports 'base64'

    ### RsaPssSignerCapability (scheme: 'rsa-pss')
    - canSign: scheme === 'rsa-pss' && privateKey exists -> true
    - sign: crypto.sign('sha256', data, {key, padding: RSA_PKCS1_PSS_PADDING, saltLength}) -> base64 signature

    ### EcdsaSignBytesCapability (scheme: 'ecdsa-secp256k1')
    - canSign: scheme === 'ecdsa-secp256k1' && privateKey exists -> true
    - sign: hashData=true (default) -> keccak256 then sign, hashData=false -> sign raw bytes
    - sign: returns hex signature string

    ### Ed25519SignBytesCapability (scheme: 'ed25519')
    - canSign: scheme === 'ed25519' && privateKey (Uint8Array) exists -> true
    - sign: @solana/kit or nacl/noble ed25519 sign -> Uint8Array signature
  </behavior>
