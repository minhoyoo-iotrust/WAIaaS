Zod OpenAPI 스키마 정의:

- ExternalActionDetailSchema: ExternalActionItemSchema.extend({ metadata, bridgeMetadata, error, txHash })
- ExternalActionsListResponseSchema: { actions: ExternalActionItemSchema[], total, limit, offset }

2개 OpenAPI 라우트:

GET /wallets/{walletId}/actions:
  - query: { venue?: string, status?: string, limit?: number(default 20, max 100), offset?: number(default 0) }
  - Drizzle 쿼리: transactions where walletId=:id AND action_kind IN ('signedData','signedHttp')
  - venue 필터: AND venue = :venue
  - status 필터: AND status = :status (또는 bridgeStatus = :status)
  - ORDER BY createdAt DESC, LIMIT/OFFSET
  - total: COUNT(*) with same filters
  - 응답에 metadata를 JSON.parse 후 provider/actionName 필드 추출

GET /wallets/{walletId}/actions/{actionId}:
  - transactions where id=:actionId AND walletId=:walletId AND action_kind IN ('signedData','signedHttp')
  - 없으면 WAIaaSError('ACTION_NOT_FOUND')
  - metadata JSON 파싱하여 전체 payload 포함
  - bridgeMetadata JSON 파싱

export function externalActionRoutes(deps): OpenAPIHono
