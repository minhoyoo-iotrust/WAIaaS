<objective>
Implement BATCH atomic execution as a single UserOperation, lazy deployment with initCode on first transaction, and ActionProvider output conversion for smart accounts.

Purpose: BATCH atomicity is the primary value proposition of smart accounts over EOA (APPROVE+SWAP in single UserOp instead of separate txs). Lazy deployment enables CREATE2 addresses to receive funds before first use. ActionProvider conversion enables DeFi protocols to work seamlessly with smart accounts.

Output: BATCH → single UserOp with calls[], deployed status update after first tx, ActionProvider → UserOp conversion. Tests covering all scenarios.

1. In `buildUserOpCalls` (from plan 315-02), implement the BATCH case:

```typescript
case 'BATCH': {
  const req = request as BatchRequest;
  return req.instructions.map((instr) => {
    // Classify by field presence (same logic as buildByType)
    if ('spender' in instr) {
      // APPROVE instruction
      const a = instr as { spender: string; token: { address: string; decimals: number }; amount: string };
      return {
        to: a.token.address as `0x${string}`,
        value: 0n,
        data: encodeFunctionData({
          abi: ERC20_TRANSFER_ABI,  // minimal ABI with approve function
          functionName: 'approve',
          args: [a.spender as `0x${string}`, BigInt(a.amount)],
        }),
      };
    }
    if ('token' in instr) {
      // TOKEN_TRANSFER instruction
      const t = instr as { to: string; amount: string; token: { address: string } };
      return {
        to: t.token.address as `0x${string}`,
        value: 0n,
        data: encodeFunctionData({
          abi: ERC20_TRANSFER_ABI,
          functionName: 'transfer',
          args: [t.to as `0x${string}`, BigInt(t.amount)],
        }),
      };
    }
    if ('calldata' in instr) {
      // CONTRACT_CALL instruction (also used by ActionProvider resolve() output)
      const c = instr as { to: string; calldata: string; value?: string };
      return {
        to: c.to as `0x${string}`,
        value: BigInt(c.value ?? '0'),
        data: (c.calldata || '0x') as `0x${string}`,
      };
    }
    // TRANSFER instruction (native transfer)
    const t = instr as { to: string; amount: string };
    return {
      to: t.to as `0x${string}`,
      value: BigInt(t.amount),
      data: '0x' as `0x${string}`,
    };
  });
}
```

```typescript
// After waitForUserOperationReceipt succeeds:
// Update deployed status if this was the first UserOp (lazy deployment)
if (ctx.wallet.accountType === 'smart') {
  // Check if wallet was undeployed before this tx
  const walletRow = await ctx.db.select().from(wallets).where(eq(wallets.id, ctx.walletId)).get();

}
```
