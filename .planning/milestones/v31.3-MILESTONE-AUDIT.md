### E2E Flow Verification

#### Flow 1: DEX Swap Quote -> Execute -- COMPLETE
1. SDK `dcentDexSwap()` -> executeAction('dcent_swap', 'dex_swap') -> REST API
2. Daemon resolves action via ActionProviderRegistry -> DcentSwapActionProvider
3. resolve('dex_swap') -> getDcentQuotes (DCent API) -> executeDexSwap
4. CAIP-19 converted to DCent IDs via currency-mapper
5. approve + txdata returned as ContractCallRequest[] BATCH
6. Pipeline processes BATCH through policy engine (CONTRACT_WHITELIST)

#### Flow 2: Exchange Quote -> Execute -> Status -- BROKEN at step 5
1. SDK `dcentExchange()` -> executeAction('dcent_swap', 'exchange') -> REST API
2. Daemon resolves action -> DcentSwapActionProvider
3. resolve('exchange') -> executeExchange -> DCent create_exchange_transaction
4. TransferRequest returned with payInAddress + exchangeMetadata
5. **BREAK**: ExchangeStatusTracker not registered with AsyncPollingService
6. **BREAK**: Status polling never starts, EXCHANGE_* notifications never emit

#### Flow 3: Auto-Routing Fallback -> 2-hop Execute -- COMPLETE
1. resolve('dex_swap') attempts direct route
2. Direct route fails with "No swap route available"
3. isNoRouteError catches -> executeTwoHopSwap fallback
4. findTwoHopRoutes probes intermediate tokens (ETH/USDC/USDT)
5. Best route selected, flat ContractCallRequest[] returned
6. Pipeline processes as BATCH

#### Flow 4: Quote Retrieval (Informational) -- COMPLETE
1. SDK `getDcentQuotes()` -> executeAction('dcent_swap', 'get_quotes')
2. DcentSwapActionProvider.resolve('get_quotes') -> queryQuotes()
3. getDcentQuotes separates DEX/exchange providers
4. Result returned with bestDexProvider recommendation
