## Out of Scope

| Feature | Reason |
|---------|--------|
| 실제 코드 구현 | v1.0은 계획 마일스톤 — 코드 작성은 v1.1부터 |
| 새로운 기능 설계 | v0.1~v0.10에서 설계 완료 — v1.0은 기존 설계를 구현 계획으로 전환 |
| 설계 문서 수정 | 구현 중 발견되는 설계 변경은 v1.1+ 인라인 수정으로 처리 |
| CI/CD 파이프라인 구축 | v1.7에서 구현 |
