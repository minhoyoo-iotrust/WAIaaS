Create `packages/actions/src/providers/jito-staking/jito-stake-pool.ts`:

Build SPL Stake Pool instructions locally using zero external dependencies (Node.js `node:crypto` only). The SolanaAdapter reconstructs TransactionInstructions from `programId`, `instructionData` (base64), and `accounts` -- so the provider must supply all resolved addresses.

**Implementation details:**

a) **Base58 encoder/decoder** (~30 lines): Standard Bitcoin base58 alphabet. `base58Decode(str): Uint8Array` and `base58Encode(bytes): string`.

b) **Ed25519 on-curve check** using Node.js 22 `crypto.subtle`:
   ```typescript
   async function isOnCurve(point: Uint8Array): Promise<boolean> {
     try {
       await crypto.subtle.importKey('raw', point, { name: 'Ed25519' }, false, ['verify']);
       return true;
     } catch {
       return false;
     }
   }
   ```

c) **`findProgramAddress(seeds, programId)`** (async):
   - Loop bump from 255 to 0
   - SHA-256 hash of [seeds..., bump_byte, programId_bytes, "ProgramDerivedAddress"]
   - Check if result is NOT on Ed25519 curve via `isOnCurve()`
   - Return [address_bytes, bump]

d) **`getAssociatedTokenAddress(wallet, mint, tokenProgram)`** -- derives ATA PDA:
   - Seeds: [wallet_bytes, token_program_bytes, mint_bytes]
   - Program: ATA_PROGRAM_ID (`ATokenGPvbdGVxr1b2hvZbsiqW5xWH25efTNsLJA8knL`)

e) **`encodeDepositSolData(amountLamports)`** -- returns base64 string:
   - Buffer: [14 (u8), amount (LE u64)]

f) **`encodeWithdrawSolData(amountLamports)`** -- returns base64 string:
   - Buffer: [16 (u8), amount (LE u64)]

g) **`buildDepositSolRequest(config, amountLamports, walletAddress)`** -- returns ContractCallRequest:
   - Derives wallet's JitoSOL ATA via `getAssociatedTokenAddress`
   - Builds all 10 account metas for DepositSol
   - Returns { type: 'CONTRACT_CALL', to: stakePoolAddress, programId, instructionData, accounts }

h) **`buildWithdrawSolRequest(config, amountLamports, walletAddress)`** -- returns ContractCallRequest:
   - Derives wallet's JitoSOL ATA
   - Builds account metas for WithdrawSol
   - Returns { type: 'CONTRACT_CALL', to: stakePoolAddress, programId, instructionData, accounts }

i) **`parseSolAmount(amount: string): bigint`** -- SOL has 9 decimals (not 18 like ETH).
   - "1.5" -> 1500000000n (1.5 SOL in lamports)
   - Implement similar to Lido's parseEthAmount but with 9 decimal places.
