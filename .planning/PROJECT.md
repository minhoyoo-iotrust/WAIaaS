# WAIaaS: AI 에이전트를 위한 Wallet-as-a-Service

## 이것이 무엇인가

AI 에이전트가 자율적으로 암호화폐 자산을 소유하고 거래할 수 있는 클라우드 지갑 서비스. 기존 WaaS가 사람 사용자를 가정한 것과 달리, AI 에이전트의 특성(API 기반 상호작용, 자율적 의사결정, 프로그래매틱 접근)에 최적화된 새로운 형태의 지갑 인프라를 제공한다.

## 핵심 가치

**AI 에이전트가 안전하고 자율적으로 온체인 거래를 수행할 수 있어야 한다** — 동시에 에이전트 주인(사람)이 자금 통제권을 유지하면서.

## 컨텍스트

### 배경

기존 Wallet-as-a-Service 솔루션들(Fireblocks, Dfns, Turnkey, Privy 등)은 다음을 가정한다:
- 사용자가 사람임 (생체인증, OTP, 이메일 인증 등)
- UI를 통한 상호작용
- 수동적 승인 흐름

AI 에이전트 시대에는 다른 접근이 필요하다:
- 프로그래매틱 인증 (API 키, 서명 등)
- 자율적 거래 실행
- 사전 정의된 규칙 기반 승인

### 에이전트 지갑 사용 모델

1. **에이전트 자율 거래**: 에이전트가 자체 지갑을 소유하고 독립적으로 거래
2. **주인 통제권**: 에이전트 주인의 지갑 주소가 등록됨
   - 주인 → 에이전트: 자금 충전 (용돈)
   - 에이전트 → 주인: 자금 회수
3. **권한 제어**: 거래 한도, 허용된 작업 유형 등 (상세 설계 필요)

### 서비스 형태

- 프레임워크에 무관한 범용 API 서비스
- 기존 에이전트 빌딩 도구와 쉬운 통합
- SDK 또는 MCP(Model Context Protocol) 형태의 통합 지원 검토

## 제약사항

- **블록체인**: Solana 메인넷 우선 — 빠른 속도, 낮은 수수료, 활발한 에이전트 생태계
- **마일스톤 범위**: 리서치 & 기획 — 구현은 다음 마일스톤
- **언어**: 모든 기획 문서 한글 작성
- **의사결정 방식**: 질문 최소화, 직접 판단하여 최선의 방법 제시
