   - accepts 3-char ISO codes: "USD", "EUR", "BTC"
   - accepts 40-char hex codes: "0158415500000000000000000000000000000000" (exactly 40 hex chars)
   - rejects "XRP" (reserved for native)
   - rejects empty string, 2-char codes, 4-char non-hex codes
   - rejects 39-char and 41-char hex codes

2. `normalizeCurrencyCode`:
   - uppercases 3-char codes: "usd" -> "USD"
   - uppercases 40-char hex codes
   - passes through already-uppercase codes

3. `parseTrustLineToken`:
   - parses "USD.rHb9CJAWyB4rj91VRWn96DkukG4bwdtyTh" -> { currency: "USD", issuer: "rHb9CJAWyB4rj91VRWn96DkukG4bwdtyTh" }
   - parses 40-char hex currency with issuer
   - throws for invalid format (no dot, empty currency, empty issuer)

4. `iouToSmallestUnit` / `smallestUnitToIou`:
   - "100.5" with 15 decimals -> 100500000000000000n
   - "0.000000000000001" with 15 decimals -> 1n
   - roundtrip: smallestUnitToIou(iouToSmallestUnit("100.5", 15), 15) === "100.5"
   - handles negative values (Trust Line balances can be negative)
   - handles "0" correctly

No mocks needed -- these are pure utility functions. Import directly from `../currency-utils.js`.
