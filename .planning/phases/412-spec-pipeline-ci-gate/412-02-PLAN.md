<task type="auto">
  <name>Task 1: Turbo 파이프라인에 generate:api-types 태스크 추가</name>
  <files>turbo.json, packages/admin/package.json</files>
  <action>
1. `turbo.json`에 `generate:api-types` 태스크 추가:
   ```json
   "generate:api-types": {
     "dependsOn": ["@waiaas/daemon#build"],
     "outputs": ["packages/admin/src/api/types.generated.ts", "packages/admin/openapi.json"],
     "cache": false
   }
   ```
   - `cache: false` -- STATE.md 블로커에 명시된 대로 stale openapi.json 방지.
   - daemon build에 의존 -- createApp import가 daemon TypeScript 소스를 직접 참조하므로, daemon 빌드가 아닌 소스 레벨 import (tsx 실행)이라 실제로는 daemon build 불필요. 하지만 daemon의 의존 패키지(@waiaas/core, @waiaas/actions 등)가 빌드되어야 import resolve 가능하므로 `"^build"` 의존성 사용:
   ```json
   "generate:api-types": {
     "dependsOn": ["^build"],
     "outputs": ["packages/admin/src/api/types.generated.ts", "packages/admin/openapi.json"],
     "cache": false
   }
   ```

2. `@waiaas/admin#build`의 dependsOn에 `"generate:api-types"` 추가:
   ```json
   "@waiaas/admin#build": {
     "dependsOn": ["@waiaas/shared#build", "generate:api-types"],
     "outputs": ["dist/**"]
   }
   ```
   이로써 admin 빌드 전에 항상 최신 타입이 생성됨.

3. root `package.json`의 scripts에 generate:api-types가 있는지 확인 (Plan 01에서 추가). Turbo는 root 스크립트를 실행하므로 문제없음.

NOTE: generate:api-types는 root-level 태스크 (패키지가 아닌 루트에서 실행). turbo.json에서 `"//#generate:api-types"` 형식으로 등록해야 루트 태스크로 인식됨:
   ```json
   "//#generate:api-types": {
     "dependsOn": [],
     "outputs": ["packages/admin/src/api/types.generated.ts", "packages/admin/openapi.json"],
     "cache": false
   }
   ```
   그리고 admin#build dependsOn에 `"//#generate:api-types"` 추가.
  </action>
  <verify>
    <automated>cd /Users/minho.yoo/dev/wallet/WAIaaS && node -e "const t=require('./turbo.json'); const genTask=t.tasks['//#generate:api-types']; if(!genTask) { console.error('missing //#generate:api-types task'); process.exit(1); } if(genTask.cache!==false) { console.error('cache must be false'); process.exit(1); } const adminBuild=t.tasks['@waiaas/admin#build']; if(!adminBuild.dependsOn.includes('//#generate:api-types')) { console.error('admin build must depend on generate:api-types'); process.exit(1); } console.log('OK')"</automated>
