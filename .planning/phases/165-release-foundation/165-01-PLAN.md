  <instructions>
1. https://www.npmjs.com 에 로그인한다
2. 다음 중 하나를 수행한다:
   - **방법 A (Organization):** https://www.npmjs.com/org/create 에서 "waiaas" 조직을 생성한다 (무료 플랜)
   - **방법 B (User scope):** npm 사용자명이 "waiaas"인 경우 자동으로 @waiaas scope를 사용할 수 있다
3. 터미널에서 확인:
   - `npm login` 으로 로그인 (이미 로그인되어 있으면 생략)
   - `npm org ls waiaas` 로 조직 멤버 확인 (Organization인 경우)
   - 또는 `npm access ls-packages @waiaas` 로 scope 접근 확인
4. scope 확보가 확인되면 다음 명령으로 dry-run 테스트:
