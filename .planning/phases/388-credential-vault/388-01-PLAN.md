  <behavior>
    --- credential-crypto ---
    - deriveCredentialKey(masterPassword) returns 32-byte Buffer (different from deriveSettingsKey for domain separation)
    - encryptCredential(plaintext, masterPassword, aad) returns {encryptedValue: Buffer, iv: Buffer, authTag: Buffer}
    - decryptCredential(encrypted, masterPassword, aad) returns plaintext string
    - Decrypt with wrong password throws Error
    - Decrypt with wrong AAD throws Error
    - AAD format: "{credentialId}:{walletId|global}:{type}"

    --- LocalCredentialVault ---
    - create(null, params) creates global credential (walletId=null), returns CredentialMetadata without value
    - create(walletId, params) creates per-wallet credential, returns CredentialMetadata
    - create with duplicate name for same walletId throws conflict error
    - get(uuid) returns DecryptedCredential with decrypted value
    - get(uuid) for non-existent id throws CREDENTIAL_NOT_FOUND
    - get(uuid) for expired credential throws CREDENTIAL_EXPIRED
    - get("walletId:name", walletId) resolves by wallet+name, per-wallet takes priority over global
    - get("name", walletId) with only global match returns global credential
    - list(walletId) returns per-wallet credentials metadata (no value)
    - list(undefined) returns global credentials metadata
    - delete(ref) removes credential
    - rotate(ref, newValue) re-encrypts with new IV, returns updated metadata
    - All metadata returned never includes value field
  </behavior>
  <action>
    1. **packages/core/src/schemas/credential.schema.ts** -- Zod SSoT:
       - CredentialTypeEnum: z.enum(['api-key', 'hmac-secret', 'rsa-private-key', 'session-token', 'custom'])
       - CreateCredentialParamsSchema: { type, name, value, metadata?, expiresAt? }

    2. **packages/daemon/src/infrastructure/credential/credential-crypto.ts**:
       - HKDF salt: "credential-vault", info: "waiaas-credential-encryption" (per doc-81 D3.6, domain separation from settings-crypto)
       - deriveCredentialKey(masterPassword: string): Buffer -- HKDF-SHA256 -> 32 bytes
       - encryptCredential(plaintext: string, masterPassword: string, aad: string): { encryptedValue: Buffer, iv: Buffer, authTag: Buffer }
         - Random 12-byte IV, AES-256-GCM, AAD bound
         - Zero key from memory after use
       - decryptCredential(data: { encryptedValue: Buffer, iv: Buffer, authTag: Buffer }, masterPassword: string, aad: string): string
         - Decrypt with GCM auth verification, zero key after use

    3. **packages/daemon/src/infrastructure/credential/credential-vault.ts**:
       - ICredentialVault interface: create, get, list, delete, rotate
       - LocalCredentialVault class:
         - Constructor: (db, getMasterPassword: () => string) -- lazy master password resolution
         - create(): UUID v7 id, encrypt with AAD "{id}:{walletId|global}:{type}", INSERT into walletCredentials
         - get(ref, walletId?): Parse ref -- if UUID format direct lookup, else try "walletId:name" split then name-only fallback
           - Per-wallet priority: query walletId+name first, then global name, then CREDENTIAL_NOT_FOUND
           - Check expiresAt < now() -> throw CREDENTIAL_EXPIRED error code
           - Decrypt and return DecryptedCredential
         - list(walletId?): SELECT metadata only (no encrypted fields), filter by walletId or IS NULL
         - delete(ref): DELETE by id
         - rotate(ref, newValue): Get existing, new IV + encrypt, UPDATE
       - Use WAIaaSError with existing error codes: CREDENTIAL_NOT_FOUND, CREDENTIAL_EXPIRED (from Phase 386)
