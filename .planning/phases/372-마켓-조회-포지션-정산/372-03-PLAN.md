   - Define `PositionDb` interface: { getPositions(walletId): rows[], getPosition(walletId, tokenId): row, upsert(row): void, updateResolution(conditionId, winningOutcome): void }
   - Constructor: `(db: PositionDb, marketData: PolymarketMarketData)`
   - `getPositions(walletId: string): Promise<PolymarketPosition[]>` -- read DB rows, enrich with current price from MarketData.getMarket(conditionId).tokens.find(t => t.token_id === row.token_id).price
   - `getPosition(walletId: string, tokenId: string): Promise<PolymarketPosition | null>`

   - `markResolved(conditionId: string, winningOutcome: string): Promise<void>` -- UPDATE polymarket_positions SET market_resolved=1, winning_outcome=$1 WHERE condition_id=$2
   - PolymarketPosition type: { id, walletId, conditionId, tokenId, marketSlug, outcome, size, avgPrice, currentPrice, unrealizedPnl, realizedPnl, marketResolved, winningOutcome, isNegRisk }

   - Pure functions, no side effects (stateless).
   - `calculateUnrealized(size: string, avgPrice: string, currentPrice: string): string` -- (currentPrice - avgPrice) * size, all in string bigint math with 6 decimal scaling
   - `calculateRealized(realizedPnl: string): string` -- passthrough (already accumulated in DB)
   - `summarize(positions: PolymarketPosition[]): PnlSummary` -- aggregate { totalUnrealized, totalRealized, byPosition: [{conditionId, outcome, unrealized, realized}] }
   - Use integer arithmetic: parse string to bigint scaled by 10^6, compute, format back to string
