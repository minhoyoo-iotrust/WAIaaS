4. Export encoding functions:
   - `encodeSupplyCalldata(asset: string, amount: bigint, onBehalfOf: string, referralCode = 0): string`
     Returns: `selector + addressToHex(asset) + uint256ToHex(amount) + addressToHex(onBehalfOf) + uint256ToHex(BigInt(referralCode))`
   - `encodeBorrowCalldata(asset: string, amount: bigint, onBehalfOf: string, interestRateMode = 2n, referralCode = 0): string`
     Returns: `selector + addressToHex(asset) + uint256ToHex(amount) + uint256ToHex(interestRateMode) + uint256ToHex(BigInt(referralCode)) + addressToHex(onBehalfOf)`
     NOTE: Parameter ORDER matters: (asset, amount, interestRateMode, referralCode, onBehalfOf) -- verify against IPool.sol
   - `encodeRepayCalldata(asset: string, amount: bigint, onBehalfOf: string, interestRateMode = 2n): string`
     Returns: `selector + addressToHex(asset) + uint256ToHex(amount) + uint256ToHex(interestRateMode) + addressToHex(onBehalfOf)`
   - `encodeWithdrawCalldata(asset: string, amount: bigint, to: string): string`
     Returns: `selector + addressToHex(asset) + uint256ToHex(amount) + addressToHex(to)`
   - `encodeApproveCalldata(spender: string, amount: bigint): string`
     Returns: `selector + addressToHex(spender) + uint256ToHex(amount)`
   - `encodeGetUserAccountDataCalldata(user: string): string`
     Returns: `selector + addressToHex(user)`
   - `encodeGetReserveDataCalldata(asset: string): string`
     Returns: `selector + addressToHex(asset)`
