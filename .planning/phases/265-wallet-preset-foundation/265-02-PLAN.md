2. **wallets.ts 변경 (PUT /wallets/:id/owner 핸들러):**

   a. import 추가: `import { BUILTIN_PRESETS } from '@waiaas/core';`

   b. body 파싱 직후 (ownerAddress 검증 전후), wallet_type 처리 로직을 추가:

   ```typescript
   // wallet_type processing
   let warning: string | null = null;
   const walletType = body.wallet_type ?? null;

   if (walletType) {
     const preset = BUILTIN_PRESETS[walletType];
     // preset은 Zod 검증 통과했으므로 반드시 존재 (WalletPresetTypeSchema가 빌트인만 허용)

     // wallet_type + approval_method 동시 제공 시 프리셋이 우선 적용 + warning
     if (body.approval_method !== undefined && body.approval_method !== null) {
       warning = `wallet_type '${walletType}' preset overrides approval_method. Preset value '${preset.approvalMethod}' applied instead of '${body.approval_method}'.`;
     }

     // 프리셋의 approval_method를 DB에 저장 (body.approval_method 대신)
     deps.sqlite.prepare(
       'UPDATE wallets SET owner_approval_method = ?, wallet_type = ? WHERE id = ?',
     ).run(preset.approvalMethod, walletType, walletId);
   } else {
     // wallet_type 미제공: 기존 approval_method 로직 유지
     // wallet_type은 그대로 두거나 NULL
   }
   ```

   c. 기존 `approval_method` 업데이트 블록을 조건부로 변경:
   - `walletType`이 있으면 위에서 이미 처리했으므로 기존 approval_method 블록을 스킵.
   - `walletType`이 없으면 기존 로직 그대로 실행.

   구체적으로:
   ```typescript
   // Update approval_method: only if wallet_type NOT provided (preset handles it above)
   if (!walletType && body.approval_method !== undefined) {
     deps.sqlite.prepare(
       'UPDATE wallets SET owner_approval_method = ? WHERE id = ?',
     ).run(body.approval_method, walletId);
   }
   ```

   d. 응답 JSON에 `walletType`과 `warning` 필드를 추가:
   ```typescript
   return c.json(
     {
       id: updated!.id,
       name: updated!.name,
       chain: updated!.chain,
       network: updated!.defaultNetwork!,
       environment: updated!.environment!,
       publicKey: updated!.publicKey,
       status: updated!.status,
       ownerAddress: updated!.ownerAddress,
       ownerVerified: updated!.ownerVerified,
       approvalMethod: updated!.ownerApprovalMethod ?? null,
       walletType: updated!.walletType ?? null,
       warning,
       updatedAt: updated!.updatedAt ? Math.floor(updated!.updatedAt.getTime() / 1000) : null,
     },
     200,
   );
   ```

   NOTE: `updated!.walletType`은 Drizzle schema의 walletType 필드를 통해 읽힌다 (Plan 01에서 추가됨).
