1. **Cargo.toml**: `tauri-plugin-updater = "2"` 의존성 추가

2. **main.rs**: `tauri::Builder`에 `.plugin(tauri_plugin_updater::Builder::new().build())` 추가. 기존 plugin 등록 체인에 이어서 추가한다.

3. **tauri.conf.json**: `bundle` 섹션에 아래 updater 설정 추가:
   ```json
   "createUpdaterArtifacts": "v1Compatible"
   ```
   최상위에 `plugins` 섹션 추가:
   ```json
   "plugins": {
     "updater": {
       "endpoints": [
         "https://github.com/minhoyoo-iotrust/WAIaaS/releases/latest/download/latest.json"
       ],
       "pubkey": "PLACEHOLDER_REPLACE_WITH_REAL_KEY"
     }
   }
   ```
   주의: pubkey는 실제 키 생성 후 교체 필요 — CI에서 `TAURI_SIGNING_PRIVATE_KEY` 환경변수로 서명.

4. **capabilities/default.json**: permissions 배열에 `"updater:default"` 추가

5. **generate-updater-key.sh**: Ed25519 키 페어 생성 헬퍼 스크립트 작성:
   ```bash
   #!/bin/bash
   # Generates Tauri updater Ed25519 key pair
   # Usage: ./generate-updater-key.sh
   # Output: prints pubkey (add to tauri.conf.json) and privkey (add to GitHub Secrets as TAURI_SIGNING_PRIVATE_KEY)
   pnpm tauri signer generate -w ~/.tauri/waiaas-desktop.key
   echo ""
   echo "=== Setup Instructions ==="
   echo "1. Copy the PUBLIC key above into tauri.conf.json plugins.updater.pubkey"
   echo "2. Add the PRIVATE key content as GitHub Secret: TAURI_SIGNING_PRIVATE_KEY"
   echo "3. If you set a password, also add it as: TAURI_SIGNING_PRIVATE_KEY_PASSWORD"
   ```
   `chmod +x` 실행 권한 부여.

6. **macOS 번들 설정**: tauri.conf.json `bundle` 섹션에 macOS 관련 추가:
   ```json
   "macOS": {
     "minimumSystemVersion": "10.15",
     "signingIdentity": null,
     "providerShortName": null
   }
   ```
   `signingIdentity`는 CI에서 `APPLE_SIGNING_IDENTITY` 환경변수로 오버라이드.

7. **Windows 번들 설정**: `bundle` 섹션에 Windows NSI 설정:
   ```json
   "windows": {
     "certificateThumbprint": null,
     "digestAlgorithm": "sha256"
   }
   ```
