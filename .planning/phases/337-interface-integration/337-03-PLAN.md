       - Section 2: NFT Query (sessionAuth)
         - GET /v1/wallet/nfts?network={network} -- list NFTs with pagination (cursor, limit)
         - Query params: network (required), cursor, limit, groupBy=collection
         - Response: { nfts: [...], cursor?, hasMore }
         - GET /v1/wallet/nfts/{tokenIdentifier}?network={network} -- individual metadata

         - Note: Returns INDEXER_NOT_CONFIGURED error if indexer API key not set
       - Section 3: NFT Transfer (sessionAuth)
         - POST /v1/transactions/send with type=NFT_TRANSFER
         - Request body: { type: "NFT_TRANSFER", to, token: { address, tokenId, standard }, network, amount? }
         - amount defaults to "1", relevant for ERC-1155 multi-copy NFTs
         - Standard values: "erc721", "erc1155", "metaplex"
         - Examples for each standard
         - Default approval tier: APPROVAL (owner must approve unless tier overridden)
         - Policies: RATE_LIMIT (nft_count counter), CONTRACT_WHITELIST
       - Section 4: NFT Approvals (sessionAuth)
         - POST /v1/transactions/send with type=APPROVE and nft field
         - Single NFT approve: { type: "APPROVE", to: spenderAddress, token: { address: contractAddress }, nft: { tokenId, standard }, network, amount: "0" }
         - Collection-wide: amount != "0" triggers setApprovalForAll (ERC-721/1155) or delegate (Solana)
         - Query approvals: GET /v1/wallet/nfts/{tokenIdentifier}/approvals?network=...&operator=...
       - Section 5: MCP Tools
         - list_nfts: params (network, cursor?, limit?, group_by?, wallet_id?)
         - get_nft_metadata: params (token_identifier, network)
         - transfer_nft: params (to, token_address, token_id, standard, network, amount?, wallet_id?)
       - Section 6: SDK Methods
         - client.listNfts({ network, cursor?, limit?, groupBy? })
         - client.getNftMetadata(tokenIdentifier, { network })
         - client.transferNft({ to, token: { address, tokenId, standard }, network, amount? })
       - Section 7: Error Codes table (NFT_NOT_FOUND 404, INDEXER_NOT_CONFIGURED 400, UNSUPPORTED_NFT_STANDARD 400, INDEXER_API_ERROR 502, NFT_METADATA_FETCH_FAILED 502)
