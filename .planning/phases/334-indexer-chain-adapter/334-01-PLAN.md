  <action>
    Create `packages/daemon/src/infrastructure/nft/alchemy-nft-indexer.ts`:

    1. `AlchemyNftIndexer` class implementing `INftIndexer`:
       - Constructor takes `{ apiKey: string, baseUrl?: string }`. Default baseUrl: `https://eth-mainnet.g.alchemy.com/nft/v3`.
       - `provider = 'alchemy'`, `supportedChains = ['ethereum']`.
       - Network-to-Alchemy-base-URL mapping: `ethereum-mainnet` -> `eth-mainnet`, `ethereum-sepolia` -> `eth-sepolia`, `polygon-mainnet` -> `polygon-mainnet`, `arbitrum-mainnet` -> `arb-mainnet`, `optimism-mainnet` -> `opt-mainnet`, `base-mainnet` -> `base-mainnet`. Use pattern `https://{chain}.g.alchemy.com/nft/v3/{apiKey}`.

    2. `listNfts(options)`:
       - Call `GET /{apiKey}/getNFTsForOwner?owner={owner}&pageSize={pageSize}&pageKey={pageKey}&withMetadata=true`
       - Map response `ownedNfts[]` to `NftItem[]`: normalize `contract.address`, `tokenId`, `tokenType` -> NftStandard, `name`/`image`/`description` from `raw.metadata`, `balance` as amount (ERC-1155).
       - Generate `assetId` using `nftAssetId()` from @waiaas/core.
       - Return `{ items, pageKey: response.pageKey, totalCount: response.totalCount }`.

    3. `getNftMetadata(network, contractAddress, tokenId)`:
       - Call `GET /{apiKey}/getNFTMetadata?contractAddress={addr}&tokenId={tokenId}`
       - Map response to `NftMetadata` including `attributes` from `raw.metadata.attributes`.

    4. `getNftsByCollection(network, collectionAddress, pageKey?)`:
       - Call `GET /{apiKey}/getNFTsForCollection?contractAddress={addr}&startToken={pageKey}&withMetadata=true`
       - Map to same `NftListResult` format.

    5. Internal `_fetch(url)` method uses global `fetch()`. On non-200 response, throw `WAIaaSError('INDEXER_API_ERROR', { message, statusCode })`. On network error, throw `WAIaaSError('INDEXER_API_ERROR', { message: error.message })`.

    6. Map Alchemy `tokenType` values: `'ERC721'` -> `'ERC-721'`, `'ERC1155'` -> `'ERC-1155'`. Unknown -> throw `WAIaaSError('UNSUPPORTED_NFT_STANDARD')`.

    Create `packages/daemon/src/infrastructure/nft/index.ts` barrel export.

    Tests: Mock global `fetch` with vi.fn(). Test all 3 methods with mock Alchemy JSON responses. Test error handling. Verify nftAssetId() integration.
