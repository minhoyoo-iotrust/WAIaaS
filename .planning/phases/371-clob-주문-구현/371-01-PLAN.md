  <action>
    1. Create `rate-limiter.ts`: PolymarketRateLimiter class. Constructor takes { maxRequests: number, windowMs: number }. Default 10 req/s (conservative, design doc 12.5). Uses sliding window: track request timestamps in array, prune expired entries. `acquire()` returns Promise<void> that resolves when a slot is available. If window full, calculate wait time to next slot expiry and setTimeout. Follow HyperliquidRateLimiter pattern but with simple request count (not weight-based).

    2. Create `clob-client.ts`: PolymarketClobClient class. Constructor takes (baseUrl: string, rateLimiter: PolymarketRateLimiter). Internal `_fetch(method, path, options)` wraps native fetch with rate limiting + error handling + JSON parsing.

       L1 endpoints (API Key lifecycle, use PolymarketSigner.signClobAuth for headers):
       - `createApiKey(walletAddress, signature, timestamp)`: POST /auth/api-key, returns { apiKey, secret, passphrase }
       - `deleteApiKey(walletAddress, signature, timestamp)`: DELETE /auth/api-key
       - `getApiKeys(walletAddress, signature, timestamp)`: GET /auth/api-keys

       L2 endpoints (trading, use PolymarketSigner.buildHmacHeaders):
       - `postOrder(apiCreds, walletAddress, signedOrder)`: POST /order, body = { order, owner, orderType, signature, signatureType }
       - `cancelOrder(apiCreds, walletAddress, orderId)`: DELETE /order/{orderId}
       - `cancelAll(apiCreds, walletAddress, conditionId?)`: POST /cancel-all
       - `getOrders(apiCreds, walletAddress)`: GET /data/orders
       - `getOrder(apiCreds, walletAddress, orderHash)`: GET /data/order/{hash}
       - `getTrades(apiCreds, walletAddress)`: GET /trades

       Public endpoints (no auth):
       - `getOrderbook(tokenId)`: GET /book?token_id={tokenId}
       - `getPrice(tokenId)`: GET /price?token_id={tokenId}
       - `getMidpoint(tokenId)`: GET /midpoint?token_id={tokenId}

       Error handling: Parse JSON error body, throw ChainError with POLYMARKET source.

    3. Create `index.ts`: Re-export all modules following hyperliquid/index.ts pattern. Export config, schemas, signer, clob-client, rate-limiter, order-builder.
