Output: Updated IActionProvider interface, ContractCallRequest with optional actionProvider field, ActionProviderRegistry with array support and auto-tagging, actions route with sequential pipeline, DatabasePolicyEngine with provider-trust logic.

     ```typescript
     const pipelineResults: Array<{ id: string; status: string }> = [];
     for (const contractCall of contractCalls) {

       const ctx: PipelineContext = { ...baseCtx, request: contractCall, txId: '' };
     ```
