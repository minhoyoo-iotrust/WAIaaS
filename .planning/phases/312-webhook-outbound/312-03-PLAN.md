  <action>
    1. Update `packages/daemon/src/services/webhook-service.ts`:
       - Add EventBus listener registration in constructor:
         a. `eventBus.on('transaction:completed', (e) => this.dispatch('TX_CONFIRMED', { txId: e.txId, txHash: e.txHash, walletId: e.walletId, network: e.network, type: e.type, amount: e.amount }))`
         b. `eventBus.on('transaction:failed', (e) => this.dispatch('TX_FAILED', { txId: e.txId, error: e.error, walletId: e.walletId, network: e.network, type: e.type }))`
         c. `eventBus.on('wallet:activity', (e) => { const eventMap: Record<string, string> = { TX_SUBMITTED: 'TX_SUBMITTED', SESSION_CREATED: 'SESSION_CREATED', OWNER_SET: 'OWNER_REGISTERED' }; const webhookEvent = eventMap[e.activity]; if (webhookEvent) this.dispatch(webhookEvent, { walletId: e.walletId, ...e.details }); })`
         d. `eventBus.on('kill-switch:state-changed', (e) => { if (e.state === 'SUSPENDED') this.dispatch('KILL_SWITCH_ACTIVATED', { activatedBy: e.activatedBy, previousState: e.previousState }); else if (e.state === 'ACTIVE' && e.previousState !== 'ACTIVE') this.dispatch('KILL_SWITCH_RECOVERED', { activatedBy: e.activatedBy }); })`
         e. `eventBus.on('transaction:incoming', (e) => this.dispatch('TX_SUBMITTED', { txHash: e.txHash, fromAddress: e.fromAddress, amount: e.amount, walletId: e.walletId, network: e.network, status: e.status }))`
       - NOTE: Direct invocations (WALLET_CREATED, WALLET_SUSPENDED, SESSION_REVOKED, POLICY_DENIED, MASTER_AUTH_FAILED, AUTO_STOP_TRIGGERED, NOTIFICATION_TOTAL_FAILURE) are handled via explicit dispatch() calls from the respective handlers. These can be added incrementally by the caller as needed -- WebhookService.dispatch() is already public.
       - Update destroy(): call `this.eventBus.removeAllListeners()` is NOT appropriate (it removes ALL listeners). Instead, store listener references and remove them individually:
         ```typescript
         private listeners: Array<{ event: string; fn: (...args: any[]) => void }> = [];
         // In constructor: store each listener
         // In destroy(): for each stored listener, eventBus won't have removeListener easily...
         ```
         Actually, since EventBus wraps Node.js EventEmitter but only exposes on/emit/removeAllListeners(event?), the simplest approach is to set `this.disposed = true` and guard dispatch() with `if (this.disposed) return`. The EventBus listeners will be cleaned up when daemon calls eventBus.removeAllListeners() at shutdown.
