# Plan 280-02: API Route Endpoint Deletion + Response Field Removal + OpenAPI Schema Cleanup

## Goal
Delete the 2 remaining default-network endpoints (PUT /wallets/:id/default-network, PUT /wallet/default-network), remove defaultNetwork/isDefault/defaultWalletId from all API responses and request schemas, update transaction/action resolvers to 3-param signature, clean admin routes, and strip OpenAPI schemas.

## Context

Phase 279 changed `resolveNetwork` from 4 to 3 params (removed `walletDefaultNetwork`). Phase 280-01 handles JWT/auth/sessions cleanup. This plan handles all remaining API route-level changes.

**Key changes from Phase 279:**
- `resolveNetwork(requestNetwork, environment, chain)` -- 3 params
- `getSingleNetwork` replaces `getDefaultNetwork` (EVM returns null)
- `wallets` table no longer has `default_network` column (Drizzle schema updated)
- `session_wallets` no longer has `is_default` column

**IMPORTANT:** Since `wallets.defaultNetwork` was removed from the Drizzle schema in Phase 279, any code doing `wallet.defaultNetwork` will fail typecheck. All such references must be replaced.
